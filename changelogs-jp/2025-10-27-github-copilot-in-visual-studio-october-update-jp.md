# Visual Studio の GitHub Copilot — 10月アップデート

## Overview
Visual Studio の 2025年10月アップデートでは、新しいモデルの選択肢、よりプロジェクトを把握する Copilot、そして Copilot Chat のプランニング機能が追加されます。

## Detailed Explanation
### 概要
- Visual Studio の 2025年10月アップデートでは、新しいモデルの選択肢、よりプロジェクトを把握する Copilot、そして Copilot Chat のプランニング機能が追加されます。

### ハイライト
- Visual Studio 2022 (v17.14) の GitHub Copilot の新機能は次のとおりです:
- 新しいモデル : Claude Sonnet 4.5 と Claude Haiku 4.5 が利用可能になり、Visual Studio 内でのエージェント的なワークフローに最先端の機能を提供します。
- メモリー : Copilot のメモリーにより、プロジェクトのコーディング規約やベストプラクティスを蓄積できます。チャットで検出された好みを .editorconfig 、 CONTRIBUTING.md 、 README.md に保存できるため、Copilot がよりプロジェクトを理解し、セッションをまたいで一貫性を保てます。
- プランニング : 組み込みのプランニング機能で、複雑なマルチステップのタスクに対応します。Copilot Chat は、タスクリスト、編集対象ファイル、アプローチのためのコンテキストを含む Markdown の計画ファイルを作成し、透明性と信頼性を高めます。
- 指示ファイル : .github/instructions に .instructions.md を追加して、プロジェクト標準やガイドラインを定義できます。glob パターンで対象ファイル/フォルダーを指定でき、コードベース全体に一貫してルールを適用しやすくなります。
- チャット管理 : 新しい /clear と /clearall コマンドで整理できます。/clear は現在のチャットをリセットし、/clearall は過去のスレッドをすべて削除します。
