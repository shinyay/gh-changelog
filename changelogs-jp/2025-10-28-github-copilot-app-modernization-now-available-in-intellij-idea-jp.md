# GitHub Copilot アプリモダナイゼーション：IntelliJ IDEA で利用可能に

## Overview
GitHub Copilot アプリモダナイゼーションが IntelliJ IDEA で利用可能になり、AI を活用したモダナイゼーションワークフローが Java 開発者に最も人気のある IDE の 1 つにもたらされました。

## Detailed Explanation
### 概要
- GitHub Copilot アプリモダナイゼーションが IntelliJ IDEA で利用可能になり、AI を活用したモダナイゼーションワークフローが Java 開発者に最も人気のある IDE の 1 つにもたらされました。
- このリリースにより、アプリケーションの評価、自動コード変換の適用、ビルドと依存関係のパッチ適用、サービスのコンテナ化が可能になります。そしてこれらすべてを IntelliJ IDEA から離れることなく実行できます。
- Copilot アプリモダナイゼーションにより、以下が可能になります：
- 詳細なアプリケーション評価レポートの生成。
- 自動コード変換の実行。
- 成功したビルドの確保、すべてのユニットテストの検証、特定された CVE の解決。
- クラウドデプロイメントのためのコンテナ化の効率化。
- これらの機能により、アップグレードと移行が加速され、チームは既存の Java プロジェクトをスピードと自信を持ってモダナイズできます。

### 始め方
- セットアップ手順については、GitHub Copilot アプリモダナイゼーションプラグインページをご覧ください。
- エンドツーエンドの紹介ビデオもご覧いただけます。
