# Visual Studio Code の GitHub Copilot がアップグレード

## Overview
以下の項目が VS Code Insiders にリリースされ、GitHub Universe 2025 で発表されました。

## Detailed Explanation
### 概要
- 以下の項目が VS Code Insiders にリリースされ、GitHub Universe 2025 で発表されました。

### ハイライト
- OpenAI Codex 統合：Copilot Pro+ サブスクリプションで、VS Code Insiders で OpenAI Codex にアクセスできます。
- Agent Sessions ビュー：ローカルおよびクラウドのエージェントセッションを統一されたインターフェースから管理します。
- コンテキスト分離サブエージェント：サブエージェントにより、メインチャットから独立して解決できる集中的なタスク（TDD ワークフローの実行、調査の実施など）を委任でき、コンテキスト最適化されたトークン使用が可能です。
- プランモード：ステップバイステップの実装計画を作成、改良、実行します。プランモードは、コードベースを分析し、詳細な実行計画を生成し、コーディングを開始する前に要件がカバーされているかを検証します。
- すべての新機能を完全なリリースノートでご確認ください。ハッピーコーディング！
- 注：Insiders ビルドは月次安定ビルドと並行してインストールされ、同じマシンでどちらでも自由に作業できます。Insiders ビルドは、VS Code 開発チームが日常的に使用しているものと同じであり、新機能を試してフィードバックを提供していただけることを本当に感謝しています。
- VS Code Insiders を開き、ヘルプ → 更新の確認を選択し、プロンプトが表示されたら再起動して、Insiders ビルドを更新してください。
