# Copilot を使用したプルリクエストでの変更のグループ化がパブリックプレビューで利用可能に

## Overview
Copilot がプルリクエストの変更を分析して整理するようになりました！この機能は現在パブリックプレビューで、Copilot Enterprise および Copilot Pro+ ユーザーが利用できます。

## Detailed Explanation
### 概要
- Copilot がプルリクエストの変更を分析して整理するようになりました！この機能は現在パブリックプレビューで、Copilot Enterprise および Copilot Pro+ ユーザーが利用できます。
- Copilot 変更グループは、リファクタリング、設定の更新、テストの改善など、関連する変更を自動的にグループ化することで、レビュワーが複雑なプルリクエストを理解しナビゲートするのを支援します。これにより、各変更セットの目的と影響を理解しやすくなります。
- Copilot 変更グループを使用すると、以下が可能になります：
- 各変更セットの背後にある意図を迅速に理解できます。
- 複雑なプルリクエストをより速く、より自信を持ってレビューできます。
- より小さな、あまり重要でない更新ではなく、重要な変更に集中できます。
- Copilot 変更グループを使用するには、Files changed タブの右上にある Try the new experience を選択して、新しい「Files changed」プレビューページにオプトインしてください。
