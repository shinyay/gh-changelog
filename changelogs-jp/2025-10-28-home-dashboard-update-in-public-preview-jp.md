# ホームダッシュボードのアップデートがパブリックプレビューで利用可能に

## Overview
ホームダッシュボードのコンテンツとレイアウトを改善し、ユーザーが単一のビューで最も重要な作業を確認、理解、アクションできるように設計しました。

## Detailed Explanation
### 概要
- ホームダッシュボードのコンテンツとレイアウトを改善し、ユーザーが単一のビューで最も重要な作業を確認、理解、アクションできるように設計しました。
- ホームダッシュボードのデフォルトビューには、最近のエージェントタスク、プルリクエスト、issue のカスタマイズ可能な短いリストが表示されるようになりました。プレビューが有効になると、フィードビューは github.com/feed の独自のページに移動しました。Feature preview で New Dashboard Experience を有効にすることで、ダッシュボードアップデートにアクセスできます。

### 新機能
- 新しいエージェントモジュールは、すべてのエージェントタスクのステータスを表示し、タスクセッションへのリンクを提供します。
- プルリクエストと issue モジュールは、進行中の作業に迅速に移動するために、最新のプルリクエストと issue をリストします。各モジュールには、リストをカスタマイズするフィルタリングオプションがあります。
- 新しい Feed タブは、フィードの新しいホームです。github.com/feed でも見つけることができます。フィードのコンテンツとフィルターは変更されていません。モバイルデバイスでサイトを使用する場合、ハンバーガーナビゲーションにフィードへのリンクがあります。
- ダッシュボードに新しいビジュアルと UI の更新を適用しました。
- アクションメニューにより、新しいモジュールのいずれかから作業を開始できます。issue を Copilot コーディングエージェントに割り当てたり、プルリクエストと issue を Copilot Chat で参照として開くことができます。
- issue を Copilot に割り当てると、使用するリポジトリ、ブランチ、カスタムエージェントを選択できるダイアログが表示されます。
