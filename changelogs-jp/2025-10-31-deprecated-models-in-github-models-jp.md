次のモデルは GitHub Models で廃止予定となりました。改善された機能を備えた拡張モデルを引き続き使用するには、推奨されるバージョンに移行してください：


* **Mistral\-Nemo** ユーザーは [Mistral\-small\-2503](https://github.com/marketplace/models/azureml-mistral/mistral-small-2503) に移行する必要があります。

また、**Jais\-30b\-chat** は GitHub Models で利用できなくなりました。


GitHub Models の詳細については、[ドキュメントをご覧ください](https://docs.github.com/github-models)。また、[コミュニティディスカッション](https://github.com/orgs/community/discussions/categories/models?discussions_q=is%3Aopen+category%3AModels+)にも参加できます。
