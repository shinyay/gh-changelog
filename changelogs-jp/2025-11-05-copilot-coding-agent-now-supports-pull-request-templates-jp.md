# Copilot コーディングエージェントがプルリクエストテンプレートをサポート

## Overview
Copilot コーディングエージェントは、非同期で自律的なバックグラウンドエージェントです。Copilot コーディングエージェントが作業を完了すると、変更の要約を含むプルリクエストの本文を更新します。現在、Copilot は本文を記述する際にリポジトリのプルリクエストテンプレートに従うことができます。

## Detailed Explanation
### 概要
- Copilot コーディングエージェントは、非同期で自律的なバックグラウンドエージェントです。Copilot コーディングエージェントが作業を完了すると、変更の要約を含むプルリクエストの本文を更新します。現在、Copilot は本文を記述する際にリポジトリのプルリクエストテンプレートに従うことができます。
- リポジトリに pull_request_template.md というファイルを追加して、プルリクエストテンプレートを作成します。リポジトリに複数のプルリクエストテンプレートが含まれている場合、Copilot はタスクに最適なものを選択します。
- コーディングエージェントを使用する際のベストプラクティスの詳細については、コーディングエージェントのドキュメントをご覧ください。

## Article Content (cleaned)
Copilot コーディングエージェントは、非同期で自律的なバックグラウンドエージェントです。Copilot コーディングエージェントが作業を完了すると、変更の要約を含むプルリクエストの本文を更新します。現在、Copilot は本文を記述する際にリポジトリのプルリクエストテンプレートに従うことができます。


リポジトリに `pull_request_template.md` というファイルを追加して、[プルリクエストテンプレート](https://docs.github.com/communities/using-templates-to-encourage-useful-issues-and-pull-requests/creating-a-pull-request-template-for-your-repository)を作成します。リポジトリに複数のプルリクエストテンプレートが含まれている場合、Copilot はタスクに最適なものを選択します。


コーディングエージェントを使用する際のベストプラクティスの詳細については、[コーディングエージェントのドキュメント](https://docs.github.com/copilot/concepts/agents/coding-agent/about-coding-agent)をご覧ください。
