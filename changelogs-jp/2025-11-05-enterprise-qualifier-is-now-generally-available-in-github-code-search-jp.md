# GitHub コード検索で enterprise: 修飾子が一般提供開始

## Overview
GitHub コード検索に新しい enterprise: 修飾子を追加し、エンタープライズ内のすべての組織を横断して簡単に検索できるようにしました。

## Detailed Explanation
### 概要
- GitHub コード検索に新しい enterprise: 修飾子を追加し、エンタープライズ内のすべての組織を横断して簡単に検索できるようにしました。
- 以前は、複数の組織を横断して検索するには、多くの org: 修飾子をブール演算子と手動で組み合わせる必要があり、数百または数千の組織を持つエンタープライズにとっては煩雑なプロセスでした。

### 使用方法
- 任意のコード検索クエリに enterprise: 修飾子を追加して、エンタープライズ内のすべての組織を横断して検索します。例については、ドキュメントをご覧ください。
- enterprise: 修飾子は、他の既存の検索修飾子とシームレスに連携し、GitHub コード検索のすべての修飾子と同じ制限の対象となります。
- この機能は、すべての GitHub Enterprise Cloud のお客様に対して一般提供を開始しました。

## Impact / Who's Affected
- この機能は、すべての GitHub Enterprise Cloud のお客様に対して一般提供を開始しました。

## Article Content (cleaned)
GitHub コード検索に新しい `enterprise:` 修飾子を追加し、エンタープライズ内のすべての組織を横断して簡単に検索できるようにしました。


以前は、複数の組織を横断して検索するには、多くの `org:` 修飾子を[ブール演算子](https://docs.github.com/enterprise-cloud@latest/search-github/github-code-search/understanding-github-code-search-syntax#using-boolean-operations)と手動で組み合わせる必要があり、数百または数千の組織を持つエンタープライズにとっては煩雑なプロセスでした。


### [使用方法](#how-it-works)


任意のコード検索クエリに `enterprise:` 修飾子を追加して、エンタープライズ内のすべての組織を横断して検索します。[例については、ドキュメントをご覧ください](https://docs.github.com/enterprise-cloud@latest/search-github/github-code-search/understanding-github-code-search-syntax#enterprise-qualifier)。


`enterprise:` 修飾子は、他の既存の検索修飾子とシームレスに連携し、[GitHub コード検索のすべての修飾子と同じ制限](https://docs.github.com/search-github/github-code-search/about-github-code-search#limitations)の対象となります。


この機能は、すべての GitHub Enterprise Cloud のお客様に対して一般提供を開始しました。
