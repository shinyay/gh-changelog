# レガシー Copilot 使用レポート CSV は利用できなくなりました

## Overview
以前 Copilot アクセス設定で組織管理者が利用可能だった、レガシー Copilot 使用レポート CSV を廃止しました。

## Detailed Explanation
### 概要
- 以前 Copilot アクセス設定で組織管理者が利用可能だった、レガシー Copilot 使用レポート CSV を廃止しました。
- 3か月前、レガシー使用レポートは、新しいより詳細なアクティビティレポートに置き換えられて非推奨となりました。製品内通知により、管理者に今後の廃止が通知されました。

### 変更内容
- 使用レポート（レガシー）削除：古い CSV ダウンロードは、Copilot アクセス設定から管理者が利用できなくなりました。

### 変更されない内容
- アクティビティレポート（新規）は残る：新しいアクティビティレポートは、使用レポートのすべてのメトリクスと詳細に加えて、追加の粒度とエクスポートワークフローの改善を提供します。
- データの変更なし：すべての使用データは、アクティビティレポートを介して引き続き利用可能です。

### 実行すべきこと
- 管理者は、組織全体またはエンタープライズ全体の Copilot 使用データを取得するために、Copilot アクティビティレポートの使用を開始する必要があります。
- レガシーレポートに依存するワークフローや自動化がある場合は、アクティビティレポートのエンドポイントと形式を参照するように更新してください。
- Copilot アクティビティレポートへのアクセスと使用の詳細については、アクティビティレポートのドキュメントをご覧ください。

## Impact / Who's Affected
- 以前 Copilot アクセス設定で組織管理者が利用可能だった、レガシー Copilot 使用レポート CSV を廃止しました。
- 3か月前、レガシー使用レポートは、新しいより詳細なアクティビティレポートに置き換えられて非推奨となりました。
