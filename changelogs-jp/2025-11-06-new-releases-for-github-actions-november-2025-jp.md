# GitHub Actions の新リリース — 2025年11月

## Overview
特定のワークフロー実行から、最大10個のネストされた再利用可能なワークフローを使用し、合計最大50個のワークフローを呼び出すことができるようになりました。以前の制限は、それぞれ4個と20個でした。これらの新しい制限により、自動化を拡張し、コラボレーションを効率化することが容易になります。

## Detailed Explanation
### 概要
- 特定のワークフロー実行から、最大10個のネストされた再利用可能なワークフローを使用し、合計最大50個のワークフローを呼び出すことができるようになりました。以前の制限は、それぞれ4個と20個でした。これらの新しい制限により、自動化を拡張し、コラボレーションを効率化することが容易になります。
- GitHub Actions M2 搭載の macOS ランナーが一般提供開始されました。macos-latest-xlarge、macos-15-xlarge、macos-14-xlarge、または macos-13-xlarge ラベルを使用して、M2 マシンでワークフローを実行し、パフォーマンスの向上と GPU アクセラレーションを獲得できます。
- 組織で GitHub Actions を有効にしなくても、GitHub Copilot コーディングエージェントを使用できるようになり、ワークフロー自動化と AI 駆動のコーディングアシスタンスの設定により柔軟性が提供されます。
