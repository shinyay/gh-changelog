# コミットメッセージ内の @メンションの通知を削除

## Overview
不要なノイズを減らし、最も重要なことに集中できるように、コミットメッセージ内のメンションから生成される通知を削除します。以前は、コミットメッセージでユーザーやチームをメンションする（例: fixes bug for @username）と、イシューやプルリクエストと同様に通知がトリガーされていました。

## Detailed Explanation
### 概要
- 不要なノイズを減らし、最も重要なことに集中できるように、コミットメッセージ内のメンションから生成される通知を削除します。以前は、コミットメッセージでユーザーやチームをメンションする（例: fixes bug for @username）と、イシューやプルリクエストと同様に通知がトリガーされていました。
- 編集者注記（2025年11月7日）: 通知のタイプに関する用語を更新しました。

### 何が変わるのか
- 12月8日以降、ユーザーとチームは、コミットメッセージでメンションされた際に通知を受け取らなくなります。イシュー、プルリクエスト、ディスカッションでのメンションは、引き続き通知をトリガーします。
