# Claude Sonnet 3.5 が廃止、Copilot Free で Claude Haiku 4.5 が利用可能に

## Overview
2025年11月10日より、すべての GitHub Copilot エクスペリエンス（Copilot Chat、インライン編集、ask および agent モード、コード補完を含む）で Claude Sonnet 3.5 を廃止しました。

## Detailed Explanation
### 概要
- 2025年11月10日より、すべての GitHub Copilot エクスペリエンス（Copilot Chat、インライン編集、ask および agent モード、コード補完を含む）で Claude Sonnet 3.5 を廃止しました。
- この変更の一環として、大幅に強力でコーディング機能が強化された Claude Haiku 4.5 が、Copilot Free を含むすべての Copilot ユーザーが利用できるようになります。これにより、Anthropic の先進技術への無料アクセスが継続されます。
- サポートされているモデルを使用するように、ワークフローと統合を更新してください。Copilot Enterprise の管理者は、Copilot 設定のモデルポリシーを通じて代替モデルへのアクセスを有効にする必要がある場合があります。管理者として、個々の Copilot 設定を確認し、特定のモデルに対してポリシーが有効になっていることを確認することで、可用性を確認できます。有効にすると、VS Code および github.com の Copilot Chat モデルセレクターにモデルが表示されます。廃止されたモデルを削除するためのアクションは必要ありません。
- 質問や懸念がある GitHub Enterprise のお客様は、アカウントマネージャーに連絡してさらなる支援を受けることをお勧めします。
