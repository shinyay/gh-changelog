# データレジデンシーを備えた GitHub Enterprise Cloud が Visual Studio サブスクリプションをサポート

## Overview
サブスクリプションベースと使用量ベースの両方の GitHub Enterprise の Visual Studio サブスクリプションでデータレジデンシーがサポートされるようになりました。これにより、地域のデータ要件に準拠しながら、Visual Studio サブスクリプションを通じて GitHub Enterprise の機能にアクセスできます。

## Detailed Explanation
### 概要
- サブスクリプションベースと使用量ベースの両方の GitHub Enterprise の Visual Studio サブスクリプションでデータレジデンシーがサポートされるようになりました。これにより、地域のデータ要件に準拠しながら、Visual Studio サブスクリプションを通じて GitHub Enterprise の機能にアクセスできます。
- 詳細については、GitHub Enterprise を使用した Visual Studio サブスクリプションについて学ぶか、データレジデンシーのドキュメントを参照してください。

## Article Content (cleaned)
サブスクリプションベースと使用量ベースの両方の GitHub Enterprise の Visual Studio サブスクリプションでデータレジデンシーがサポートされるようになりました。これにより、地域のデータ要件に準拠しながら、Visual Studio サブスクリプションを通じて GitHub Enterprise の機能にアクセスできます。
