# Raptor mini が GitHub Copilot のパブリックプレビューで展開中

## Overview
新しい実験的モデルである Raptor mini が、Visual Studio Code の GitHub Copilot の Pro、Pro+、および Free プランで展開されています。

## Detailed Explanation
### 概要
- 新しい実験的モデルである Raptor mini が、Visual Studio Code の GitHub Copilot の Pro、Pro+、および Free プランで展開されています。
- Visual Studio Code の Copilot Chat モデルピッカーから、chat、ask、edit、および agent モードでモデルを選択できるようになります。
