# OpenAIのGPT-5.1、GPT-5.1-Codex、GPT-5.1-Codex-MiniがGitHub Copilotでパブリックプレビューに

## Overview
GPT-5.1、GPT-5.1-Codex、GPT-5.1-Codex-Mini―OpenAIの最新5.1シリーズモデルのフルスイートが、GitHub Copilotでパブリックプレビューとしてロールアウトされています。

## Detailed Explanation
### Overview
- GPT-5.1、GPT-5.1-Codex、GPT-5.1-Codex-Mini―OpenAIの最新5.1シリーズモデルのフルスイートが、GitHub Copilotでパブリックプレビューとしてロールアウトされています。

### Availability in GitHub Copilot
- OpenAI GPT-5.1、GPT-5.1-Codex、GPT-5.1-Codex-MiniはCopilot Pro、Pro+、Business、Enterpriseで利用可能になります。
- モデルによって使用領域と製品内での利用可能性が異なります：
- GPT-5.1
- Visual Studio Code：全モード（chat、ask、edit、agent）
- JetBrains：全モード（ask、edit、agent）
- Xcode：全モード（ask、agent）
- Eclipse：全モード（ask、agent）
- github.com
- GitHub Mobile（iOSおよびAndroid）
- Copilot CLI
- GPT-5.1-CodexおよびGPT-5.1-Codex-Mini
- Visual Studio Code バージョン1.104.1以降：全モード（chat、ask、edit、agent）
- GitHub Copilot for JetBrains バージョン1.5.61以降：全モード（ask、edit、agent）
- GitHub Copilot for Xcode バージョン0.45.0以降：全モード（ask、agent）
- GitHub Copilot for Eclipse バージョン0.13.0以降：全モード（ask、agent）
- Copilot CLI
- ロールアウトは段階的です。まだ新しいモデルが表示されない場合は、しばらくしてから確認してください。

### Enabling access
- EnterpriseおよびBusinessプラン：管理者はCopilot設定でOpenAI GPT-5.1、GPT-5.1-Codex、またはGPT-5.1-Codex-Miniポリシーを有効にする必要があります。有効にすると、その組織のユーザーはCopilot Chatモデルピッカーで該当するモデルを表示できます。
- ProおよびPro+プラン：ユーザーはモデルピッカーでモデルを選択し、ワンタイムプロンプトを確認することでモデルを有効にできます。
- Bring your own key：ピッカーから「Manage Models」を選択し、OpenAI GPT-5.1、GPT-5.1-Codex、またはGPT-5.1-Codex-Miniを選択し、プロンプトが表示されたらAPIキーを入力します。

- GitHub Copilotで利用可能なすべてのモデルを探索するには、モデルに関するドキュメントを参照し、Copilotを始めてください。
