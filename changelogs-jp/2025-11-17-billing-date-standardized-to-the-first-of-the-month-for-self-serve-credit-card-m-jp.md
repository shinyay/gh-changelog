# セルフサービスクレジットカード従量制Enterpriseカスタマー向け請求日の月初への標準化が一般提供に

## Overview
2025年12月1日より、セルフサービス従量制GitHub Enterprise Cloudアカウントでクレジットカード支払いを利用しているすべての使用量ベースのGitHub製品は、毎月1日に請求されます。請求期間は暦月のまま変更ありません。操作は不要で、この更新は自動的に適用されます。

## Detailed Explanation
### Overview
- 2025年12月1日より、セルフサービス従量制GitHub Enterprise Cloudアカウントでクレジットカード支払いを利用しているすべての使用量ベースのGitHub製品は、毎月1日に請求されます。請求期間は暦月のまま変更ありません。操作は不要で、この更新は自動的に適用されます。
- これは以下の従量制GitHub製品に適用されます：
- GitHub Actions（使用分数、ストレージ）
- GitHub Codespaces（コンピュート時間、ストレージ）
- GitHub Packages（ストレージ、データ転送）
- Advanced Security（アクティブコミッター席）
- Copilot Business/Enterprise（従量制シート超過分）
- 追加のストレージまたは帯域幅料金
- 情報提供のため、この今後の変更について請求概要ページにバナーを表示しています。
- 重要：11月に処理された支払いは、支払日に関係なく、10月の使用量に関連するものです。12月1日の支払いには、11月のすべての使用量が含まれます。これは重複請求ではありません。
- この変更の理由は？ – より予測可能で一貫した請求スケジュールを実現します。 – 標準的な支払日により、請求の明確性を向上させます。
- この変更をオプトアウトする場合は、請求書払いに切り替える必要があります。詳細については営業チームにお問い合わせください。

## Caveats / Limitations
- 重要：11月に処理された支払いは、支払日に関係なく、10月の使用量に関連するものです。

## Article Content (cleaned)
**2025年12月1日**より、セルフサービス従量制GitHub Enterprise Cloudアカウントでクレジットカード支払いを利用しているすべての使用量ベースのGitHub製品は、毎月1日に請求されます。請求期間は暦月のまま変更ありません。操作は不要で、この更新は自動的に適用されます。


これは以下の従量制GitHub製品に適用されます：


* GitHub Actions（使用分数、ストレージ）
* GitHub Codespaces（コンピュート時間、ストレージ）
* GitHub Packages（ストレージ、データ転送）
* Advanced Security（アクティブコミッター席）
* Copilot Business/Enterprise（従量制シート超過分）
* 追加のストレージまたは帯域幅料金


情報提供のため、この今後の変更について請求概要ページにバナーを表示しています。


**重要：**11月に処理された支払いは、支払日に関係なく、10月の使用量に関連するものです。12月1日の支払いには、11月のすべての使用量が含まれます。これは重複請求ではありません。


**この変更の理由は？**  

– より予測可能で一貫した請求スケジュールを実現します。  

– 標準的な支払日により、請求の明確性を向上させます。


この変更をオプトアウトする場合は、請求書払いに切り替える必要があります。詳細については[営業チーム](https://github.com/contact/sales)にお問い合わせください。
