# Copilot使用状況メトリクスの詳細な権限が利用可能に

## Overview
エンタープライズロール：Enterprise Copilotメトリクスの表示

## Detailed Explanation
### What's new?
- エンタープライズロール：Enterprise Copilotメトリクスの表示
- 新しいエンタープライズロール「Enterprise Copilotメトリクスの表示」が割り当てられている場合、Copilot使用状況メトリクスAPIとダッシュボードにアクセスできるようになりました
- この重要な拡張により、エンタープライズ管理者や請求マネージャーだけでなく、このロールを割り当てられたすべてのユーザーが、エンタープライズの詳細なCopilot使用状況メトリクスを表示できるようになりました。

### Who can now access Copilot usage metrics?
- 「Enterprise Copilotメトリクスの表示」ロールを持つすべてのユーザー
- エンタープライズ管理者と請求マネージャーは引き続きデフォルトでアクセスできます。

### How to get started
- 「AI Controls」>「Copilot」で「Copilot使用状況メトリクス」ポリシーが有効になっていることを確認してください。
- 「People」>「Enterprise roles」で、ユーザーに新しい「Enterprise Copilotメトリクスの表示」エンタープライズロールを割り当ててください。
- 新たに割り当てられたユーザーに、Copilot使用状況メトリクスAPIとダッシュボードにアクセスして、今日からインサイトを活用するようアドバイスしてください。
- Copilot使用状況メトリクスの詳細をご覧ください。
