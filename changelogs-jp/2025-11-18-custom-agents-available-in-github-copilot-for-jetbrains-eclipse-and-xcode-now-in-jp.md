# JetBrains、Eclipse、Xcode用GitHub Copilotでカスタムエージェントがパブリックプレビューに

## Overview
JetBrains IDE、Eclipse、Xcode用のカスタムエージェントがパブリックプレビューになり、プロジェクトや組織に合わせた再利用可能な専門知識、ガードレール、ツールアクセスをAIエージェントにパッケージ化できるようになりました。

## Detailed Explanation
### Overview
- JetBrains IDE、Eclipse、Xcode用のカスタムエージェントがパブリックプレビューになり、プロジェクトや組織に合わせた再利用可能な専門知識、ガードレール、ツールアクセスをAIエージェントにパッケージ化できるようになりました。

### What's new
- カスタムエージェントは、名前、説明、ツール、モデルを指定できるようにすることで、チャットモードにカスタマイズをもたらします。JetBrains、Eclipse、Xcodeでワークフローとコーディング標準に合わせた特化AIエージェントを作成できます。プロンプトを指定するMarkdownファイルを使用してエージェントを定義するため、IDEですばやく起動して実行できます。

### Try it out
- IDEを更新：JetBrains IDE、Eclipse、またはXcode用の最新のCopilot拡張機能をインストールして、カスタムエージェントプレビューにアクセスしてください。
- カスタムエージェントを作成：カスタムエージェントクイックスタートとハウツーガイドに従って、エージェントの名前、説明、指示、知識ソース、ツールポリシーを記述したスターター構成ファイルを生成してください。
- カスタムエージェントを選択：Copilot Chatで、利用可能なエージェントのリストからカスタムエージェントを選択してください。
- エージェントをテスト：現在のカスタムエージェントで会話を開始してください。

### Share your feedback
- フィードバックがロードマップを推進します。IDE内のフィードバックコントロールまたは既存のCopilotサポートチャネルを使用して、カスタムエージェントがどのように機能しているかをお知らせください。
- 製品内フィードバック：IDE内のフィードバックオプションを使用してください。
