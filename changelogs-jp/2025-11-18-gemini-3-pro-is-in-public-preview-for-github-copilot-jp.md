# Gemini 3 ProがGitHub Copilotでパブリックプレビューに

## Overview
Googleの最新フロンティアモデルGemini 3 Proがロールアウトされ、GitHub Copilot Pro、Pro+ Business、Enterpriseサブスクリプションで利用可能になります。ロールアウトは段階的です。まだ表示されない場合は、しばらくしてから確認してください。

### Enabling access
- EnterpriseおよびBusinessプラン：管理者はCopilot設定でGemini 3 Proポリシーを有効にする必要があります。有効にすると、その組織のユーザーはサポートされているサーフェスでGemini 3 Proを表示できます。
- ProおよびPro+プラン：Visual Studio Codeピッカーでモデルを選択し、ワンタイムプロンプトを確認することで有効にします。
- Bring your own key：ピッカーから「Manage Models」を選択し、Gemini 3 Proを選択し、プロンプトが表示されたらAPIキーを入力します。
