# GitHub Copilot next edit suggestions（NES）がXcodeとEclipseでパブリックプレビューに

## Overview
GitHub Copilot next edit suggestions（NES）がXcodeとEclipse向けにパブリックプレビューで利用可能になりました。この新機能は、現在のコンテキストに基づいて次の論理的な編集を提案することで、開発者がより速く作業できるようにし、反復的な手順を減らして生産性を向上させます。

## Detailed Explanation
### Overview
- GitHub Copilot next edit suggestions（NES）がXcodeとEclipse向けにパブリックプレビューで利用可能になりました。この新機能は、現在のコンテキストに基づいて次の論理的な編集を提案することで、開発者がより速く作業できるようにし、反復的な手順を減らして生産性を向上させます。

### What's new
- GitHub Copilotのnext edit suggestions（NES）機能は、最近の編集を使用して、次に変更したい内容と場所を予測します。この新機能により、Copilotはコード、コメント、テストなどの変更を提案できます。さらに、変更を即座にプレビューして適用できます。編集をすばやく進めて、コーディングフローを中断することなく維持するのは簡単です：Tabキーで提案をナビゲートし、もう一度Tabを押して受け入れます。

### Try it out
- NESはプレビュー段階で、すべてのXcodeおよびEclipseユーザーに本日から利用可能です。アクセスするには、最新のプラグインまたはアプリバージョンに更新し、有効なCopilotライセンスがあることを確認してください。
- IDE設定でNESを有効にする：Xcodeの場合：GitHub Copilot for Xcodeアプリを開き、**Advanced** > **Suggestion Settings**に移動し、**Enable next edit suggestions (NES)**をチェックします。Eclipseの場合：GitHub Copilotアイコンをクリックし、**Edit Preferences**を選択し、**Completion**セクションを見つけて、**Enable next edit suggestions (NES)**をチェックします。
- コーディングを開始して、Copilotが次の動きを予測するのを見てください！NES提案は自動的に表示されます。キャレットまたはキーボードショートカットを使用して提案に移動します。Tabキー（デフォルト）で受け入れるか、ガターメニューの矢印を使用して適用します。
- Xcodeの場合：GitHub Copilot for Xcodeアプリを開き、**Advanced** > **Suggestion Settings**に移動し、**Enable next edit suggestions (NES)**をチェックします。
- Eclipseの場合：GitHub Copilotアイコンをクリックし、**Edit Preferences**を選択し、**Completion**セクションを見つけて、**Enable next edit suggestions (NES)**をチェックします。
