# JetBrains、Eclipse、XcodeでのGitHub Copilotのプランモードがパブリックプレビューになりました

## Overview
GitHub Copilotのプランモードが、JetBrains IDE、Eclipse、Xcode向けにパブリックプレビューとして提供開始されたことをお知らせします。コーディング前に設計図戦略を提供するこの新しいCopilot機能について、知っておくべきことをご紹介します。

## Detailed Explanation
### Overview
- GitHub Copilotのプランモードが、JetBrains IDE、Eclipse、Xcode向けにパブリックプレビューとして提供開始されたことをお知らせします。コーディング前に設計図戦略を提供するこの新しいCopilot機能について、知っておくべきことをご紹介します。

### What's new
- プランモードは、開発者がIDE内で直接ステップバイステップの実装計画を作成、改良、実行できるようにします。コードベースを分析し、詳細な実行計画を生成し、コーディングを開始する前に要件がカバーされていることを検証します。
- プランモードは、計画がレビューされ承認されるまで、コードの変更を一切行いません。要約とステップの詳細を受け取った後、レビューするか、実装のためにエージェントに引き渡すことができます。その結果、プランモードはシームレスな複数ステップのタスクをサポートし、あらゆる段階で正確性と効率性を実現します。
- この機能は、JetBrains、Eclipse、Xcodeに追加しているエージェント基盤の上に構築されています。

### How to use plan mode
- JetBrains IDEの場合：1. サインインしてCopilot Chatを開き、エージェントドロップダウンからプランモードを選択します。2. 機能の追加、コードのリファクタリング、バグ修正など、高レベルのタスクを説明し、送信をクリックします。（例：「HTML、CSS、JSファイルでシンプルなToDoアプリを作成してください。」）3. 生成された計画をレビューし、ニーズに基づいて「実装開始」または「エディタで開く」を選択します。
- Eclipseの場合：1. Copilot Chatパネルにサインインし、エージェントドロップダウンからプランモードを選択します。2. 機能の追加、コードのリファクタリング、バグ修正など、高レベルのタスクを説明し、送信をクリックします。（例：「GUIと追加、削除、更新機能を備えたToDoアプリを作成してください。」）3. 生成された計画をレビューし、ニーズに基づいて「実装開始」または「エディタで開く」を選択します。
- Xcodeの場合：1. GitHub Copilot for Xcodeを開き、サインインし、エージェントドロップダウンからプランモードを選択します。2. 機能の追加、コードのリファクタリング、バグ修正など、高レベルのタスクを説明し、送信をクリックします。（例：「GUIを備えたマインスイーパーゲームを構築してください。」）3. 生成された計画をレビューし、ニーズに基づいて「実装開始」または「エディタで開く」を選択します。

### Try it out
- プランモードは現在プレビュー中で、本日からすべてのJetBrains IDE、Eclipse、Xcodeユーザーが利用できます。アクセスするには、プラグインまたはアプリを最新バージョンに更新する必要があります。これらのIDEでCopilotエージェントモードにアクセスするには、有効なCopilotライセンスも必要です。
