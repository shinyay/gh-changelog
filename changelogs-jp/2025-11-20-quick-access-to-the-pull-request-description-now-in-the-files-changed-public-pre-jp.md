# 新しい「Files changed」パブリックプレビューでプルリクエストの説明への素早いアクセスが可能になりました

## Overview
新しい「Files changed」ページから離れることなく、プルリクエストの説明を表示できるようになりました。ページの上部にスクロールしたり、タブを切り替えたり、コンテキストを失う必要はありません。

## Detailed Explanation
### Overview
- 新しい「Files changed」ページから離れることなく、プルリクエストの説明を表示できるようになりました。ページの上部にスクロールしたり、タブを切り替えたり、コンテキストを失う必要はありません。
- 説明を表示するには、ツールバーのOverviewボタンをクリックします：
- これは、集中し続け、タブ間の切り替え頻度を減らすのに役立つ、今後の多くの改善の最初のものです。
- この機能は、現在パブリックプレビュー中の新しい「Files changed」ページでのみ利用可能です。まだクラシックエクスペリエンスを使用している場合は、クラシックページの上部にある「Try the new experience」をクリックしてください。

### 🔧 Performance and more
- 前回の更新以降、パフォーマンスを含む多数の修正が行われました。たとえば、新しい「Files changed」ページから他のページへのナビゲーションが高速になるはずです。また、今後数週間で目に見えるようになる、より多くのパフォーマンス改善にも取り組んでいます。
