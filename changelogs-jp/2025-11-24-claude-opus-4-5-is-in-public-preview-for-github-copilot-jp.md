# Claude Opus 4.5がGitHub Copilotでパブリックプレビューになりました

## Overview
Anthropicの最新モデルであるClaude Opus 4.5が、GitHub Copilotで展開されています。初期テストでは、Claude Opus 4.5は内部コーディングベンチマークを上回りながら、トークン使用量を半分に削減しており、このモデルは特にプロフェッショナルな環境で開発を加速するのに適しています。

## Detailed Explanation
### Overview
- Anthropicの最新モデルであるClaude Opus 4.5が、GitHub Copilotで展開されています。初期テストでは、Claude Opus 4.5は内部コーディングベンチマークを上回りながら、トークン使用量を半分に削減しており、このモデルは特にプロフェッショナルな環境で開発を加速するのに適しています。

### Availability in GitHub Copilot
- Claude Opus 4.5は、Copilot Pro、Pro+、Business、およびEnterpriseユーザーが利用でき、2025年12月5日までプロモーション価格の1xプレミアムリクエスト乗数で提供されます。
- モデルピッカーでモデルを選択できるようになります：
- VS CodeのCopilot Chat（Agent、Plan、Ask、Editモード）
- github.comのCopilot Chat
- GitHub Mobile（iOSおよびAndroid）のCopilot Chat
- Copilot CLI
- 展開は段階的に行われます。まだ表示されない場合は、後ほどご確認ください。
- プロモーション期間中、Claude Opus 4.5は、有効化されている場合、Copilotコーディングエージェントのデフォルトモデルとしても展開されます。

### Enabling access
- EnterpriseおよびBusinessプラン：管理者はCopilot設定でClaude Opus 4.5ポリシーを有効にする必要があります。
- ProおよびPro+プラン：VS Codeのユーザーの場合、モデルピッカーでモデルを選択し、1回限りのプロンプトを確認します。

### Learn more
- GitHub Copilotで利用可能なすべてのモデルを探索するには、モデルに関するドキュメントを参照し、Copilotを開始してください。

- GitHub Communityに参加して、フィードバックを共有してください。
