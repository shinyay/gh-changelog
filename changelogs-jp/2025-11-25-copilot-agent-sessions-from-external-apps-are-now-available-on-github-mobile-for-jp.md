# 外部アプリからのCopilotエージェントセッションがGitHub Mobile for Androidで利用可能になりました

## Overview
お気に入りのAndroidアプリから直接エージェントセッションをすばやく開始できるようになりました。GitHub Mobile for Androidの新しいAndroid sharesheet custom actionにより、外部アプリからテキストを共有して、即座にCopilotエージェントセッションを作成できるようになりました。GitHub Androidアプリを使用している人は誰でも、他の場所で作成されたアイデア（メモ、メッセージ、計画など）を、新しいエージェントセッションに送信して、Copilotを動かすことができます。このアップデートにより、ワークフローが合理化され、インスピレーションをより迅速にアクションに変えることができます。次に機能をブレインストーミングしたり、プロジェクトをマッピングしたりするときに試してみてください。
