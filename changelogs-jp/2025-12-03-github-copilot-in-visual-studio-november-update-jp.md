# GitHub Copilot in Visual Studio — 11月アップデート

## Overview
2025年11月は、Visual Studio 2026の一般提供、新しいCopilotアクションなどをもたらしました。

## Detailed Explanation
### Overview
- 2025年11月は、Visual Studio 2026の一般提供、新しいCopilotアクションなどをもたらしました。

### ハイライト
- Visual Studio 2026のGitHub Copilotの新機能は以下のとおりです：
- Visual Studio 2026一般提供：Visual Studio 2026をダウンロードして、パフォーマンス、UX、AI支援ワークフローが向上した最新バージョンのVisual Studioを始めましょう。
- パブリックプレビュー中のGitHubクラウドエージェント：新しいクラウドエージェントを使用して、Visual Studioから直接反復的なタスクを委任します。設定から「Enable Copilot Coding Agent (Preview)」をチェックして、UIのクリーンアップ、リファクタリング、ドキュメントの更新、複数ファイルの編集をオフロードしながら、コア開発に集中できます。
- コンテキストメニューのCopilotアクション：コンテキストメニューで新しいCopilotアクションにアクセスできます！コードコメント、説明、最適化をワンクリックで生成できます。
- オールインワン検索のインテント検出：新しい「Did you mean」機能により、ファイルを見失うことはありません。Copilotは検索語を分析し、タイプミスやあいまい検索を検出すると、より良いマッチを提案し、必要なものをより速く見つけるのに役立ちます。

- コンテキストメニューのCopilotアクション：コンテキストメニューで新しいCopilotアクションにアクセスできます！コードコメント、説明、最適化をワンクリックで生成できます。
