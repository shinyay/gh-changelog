# OpenAIのGPT-5.1-Codex-MaxがGitHub Copilotでパブリックプレビューとしてリリースされました

## Overview
GPT-5.1-Codex-MaxがGitHub Copilotでパブリックプレビューとして展開中です。

## Detailed Explanation
### 概要
- GPT-5.1-Codex-MaxがGitHub Copilotでパブリックプレビューとして展開中です。

### GitHub Copilotでの利用可能性
- OpenAIのGPT-5.1-Codex-MaxはCopilot Pro、Pro+、Business、およびEnterpriseで利用可能になります。
- 以下の場所にあるCopilot Chatモデルピッカーからモデルを選択できます:
  - Visual Studio Codeのすべてのモード: chat、ask、edit、agent
  - github.comのCopilot Chat
  - GitHub Mobile(iOSおよびAndroid)のCopilot Chat
  - Copilot CLI
- 展開は段階的に行われます。新しいモデルがまだ表示されない場合は、後ほどご確認ください。

### アクセスの有効化
- **EnterpriseおよびBusinessプラン**: 管理者はCopilot設定でGPT-5.1-Codex-Maxポリシーを有効にする必要があります。有効にすると、その組織のユーザーはCopilot Chatモデルピッカーで該当モデルを確認できます。
- **ProおよびPro+プラン**: ユーザーはモデルピッカーでモデルを選択し、一度限りのプロンプトを確認することでモデルを有効にできます。
- **Bring your own key**: ピッカーから「Manage Models」を選択し、GPT-5.1-Codex-Maxを選択して、プロンプトが表示されたらAPIキーを入力します。

### 詳細を学ぶ
- GitHub Copilotで利用可能なすべてのモデルについては、モデルに関するドキュメントをご覧いただき、Copilotを始めましょう。
