# Copilot ProおよびPro+サブスクライバー向けのCopilotコーディングエージェント用モデルピッカー

## Overview
Copilot ProおよびPro+サブスクライバーは、非同期で自律的なバックグラウンドエージェントであるCopilotコーディングエージェントを使用する際にモデルを選択できるようになりました。

## Detailed Explanation
### 概要
- Copilot ProおよびPro+サブスクライバーは、非同期で自律的なバックグラウンドエージェントである[Copilotコーディングエージェント](https://docs.github.com/copilot/concepts/agents/coding-agent/about-coding-agent)を使用する際にモデルを選択できるようになりました。
- エージェントタブ、エージェントパネル、github.comでのissue割り当て、またはRaycast拡張機能からタスクを開始する際に、ドロップダウンを使用して以下のモデルから選択できます:
  - Auto
  - Anthropic Claude Opus 4.5
  - Anthropic Claude Sonnet 4.5
  - OpenAI GPT-5.1-Codex-Max
- 設定を**Auto**のままにすると、Copilotは可用性に基づいてモデルを自動的に選択し、速度とパフォーマンスを最適化します。
- モデルピッカーは現在、Copilot ProおよびPro+サブスクライバーに提供されており、Copilot BusinessおよびEnterpriseサブスクライバー、ならびにVisual Studio CodeやGitHub Mobileなどのより多くのCopilotコーディングエージェント統合への提供も近日公開予定です。
- 詳細については、Copilotコーディングエージェントでのモデル選択に関するドキュメントをご覧ください。

## Impact / Who’s Affected
- モデルピッカーは現在、Copilot ProおよびPro+サブスクライバーに提供されており、Copilot BusinessおよびEnterpriseサブスクライバー、ならびにVisual Studio CodeやGitHub Mobileなどのより多くのCopilotコーディングエージェント統合への提供も近日公開予定です。
