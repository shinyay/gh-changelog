# OpenAIのGPT-5.2がGitHub Copilotでパブリックプレビュー開始

## Overview
GPT-5.2はOpenAIの最新モデルで、長いコンテキストとフロントエンドUI生成に焦点を当てており、Copilot Pro、Pro+、Business、Enterpriseで利用可能になります。

## Detailed Explanation
### Overview
- GPT-5.2はOpenAIの最新モデルで、長いコンテキストとフロントエンドUI生成に焦点を当てており、Copilot Pro、Pro+、Business、Enterpriseで利用可能になります。

### GitHub Copilotでの利用可能性
- OpenAI GPT-5.2は、Copilot Pro、Pro+、Business、Enterpriseで利用可能になります。
- 以下のいずれかからCopilotモデルピッカーでモデルを選択できます：
- Visual Studio Codeバージョン1.104.1以降、すべてのモード：chat、ask、edit、agent
- github.comのCopilot Chat
- GitHub Mobile（iOSおよびAndroid）
- Copilot CLI
- ロールアウトは段階的に行われます。新しいモデルがまだ表示されない場合は、近日中に再度確認してください。

### アクセスの有効化
- EnterpriseおよびBusinessプラン：管理者はCopilot設定でOpenAI GPT-5.2を有効にする必要があります。有効にすると、その組織のユーザーはCopilotモデルピッカーに該当モデルが表示されます。
- ProおよびPro+プラン：ユーザーは、モデルピッカーでモデルを選択し、1回限りのプロンプトを確認することで、モデルを有効にできます。
- 独自のキーを使用：Visual Studio Code内で、ピッカーから「Manage Models」を選択し、「OpenAI GPT-5.2」を選択して、プロンプトが表示されたらAPIキーを入力します。

### 詳細情報
- GitHub Copilotで利用可能なすべてのモデルを確認するには、モデルに関するドキュメントを参照し、Copilotを始めてください。
