# GitHub Discussionsで管理者として投稿が可能に

## Overview
リポジトリ管理者は、「Admin」バッジを使用してGitHub Discussionsに投稿およびコメントできるようになり、公式な回答と個人的な貢献を区別できます。

## Detailed Explanation
### Overview
- リポジトリ管理者は、「Admin」バッジを使用してGitHub Discussionsに投稿およびコメントできるようになり、公式な回答と個人的な貢献を区別できます。
- 管理者として投稿するオプションは、ディスカッションの作成やコメントの追加時に表示されます。これにより、管理者は、自分の返信が公式なリポジトリガイダンスを表す場合を選択できます。これにより、コミュニティメンバーは権威ある回答を簡単に識別でき、メンテナーは個人的に会話に参加する柔軟性を保持できます。この機能により、セキュリティリスクをもたらし、アクセス管理を複雑にする一般的な回避策である、別の管理者アカウントの必要性がなくなります。

### 管理者として投稿する方法
- 管理者エイリアスを使用して投稿するには：
- リポジトリの管理者権限を持っている必要があります。
- リポジトリでGitHub Discussionsを有効にします。
- ディスカッションの作成または返信時に「Admin」を選択します。
- コミュニティのディスカッション管理について詳しく学びましょう。
- この機能は、ディスカッションが有効になっているすべてのリポジトリでgithub.comで利用可能になりました。

### [管理者として投稿する方法](#how-to-post-as-admin)


管理者エイリアスを使用して投稿するには：


1. リポジトリの管理者権限を持っている必要があります。
2. リポジトリでGitHub Discussionsを有効にします。
3. ディスカッションの作成または返信時に**Admin**を選択します。


[コミュニティのディスカッション管理](https://docs.github.com/en/discussions/managing-discussions-for-your-community/managing-discussions)について詳しく学びましょう。


この機能は、ディスカッションが有効になっているすべてのリポジトリでgithub.comで利用可能になりました。
