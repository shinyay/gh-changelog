# VNET注入ランナーの改善された診断と必須のセルフホストランナーアップグレード

## Overview
編集者注（2025年12月19日）：このポストを更新して、セルフホストランナーをv2.329.0以降に更新する必要がある新しい日付を示しました。元のアップグレード日は2026年3月に延期され、2026年2月からブラウンアウト期間を追加しました。

## Detailed Explanation
### Overview
- 編集者注（2025年12月19日）：このポストを更新して、セルフホストランナーをv2.329.0以降に更新する必要がある新しい日付を示しました。元のアップグレード日は2026年3月に延期され、2026年2月からブラウンアウト期間を追加しました。
- Azureプライベートネットワーキング（Azure VNET-injection）を使用するGitHubホストの大規模ランナーは、接続の問題が発生したときに詳細なネットワーク診断を表示するようになりました。
- 一般的なプールエラーの代わりに、以下が表示されます：
- エンドポイントごとの可視性：各必須エンドポイントをドリルダウンして、接続が失敗している正確な場所を確認します。
- 接続メトリクス：試行回数、失敗率、成功率を表示して、包括的な健全性の洞察を得ます。
- 失敗の分類：接続タイムアウト、プロキシの設定ミス、DNS解決の失敗、TLSインターセプションの問題、ブロックされたドメインなどの特定の問題を識別します。
- これにより、サポートに連絡することなく、ネットワーク構成の問題を迅速に診断して解決できます。

### セルフホストランナーの必須アップグレード
- GitHub Actionsは、CI/CDおよび開発者自動化のための信頼性、スケーラビリティ、新機能の向上を提供するために再アーキテクチャを進めています。この取り組みの一環として、古いバージョンのactionsランナーはサポートされなくなります。
- 変更内容
- GitHubは、構成中にセルフホストランナーがv2.329.0以降を使用することを要求します。レガシーバージョンを使用しているランナーは、GitHub Actionsへの接続がブロックされます。この変更により、新しいアーキテクチャとの互換性が確保され、将来のロールアウト中にワークフローの失敗が防止されます。
- 以前は、古いランナーは構成後に自動アップグレードできました。今後、ランナーは登録前に最小バージョン要件を満たす必要があります。
- 2026年2月9日にブラウンアウトを開始し、完全な施行は2026年3月2日に開始されます。
- 必要なアクション
- セルフホストランナーをできるだけ早くv2.329.0以降にアップグレードしてください。ガイダンスについては、セルフホストランナーのドキュメントを参照してください。

### ActionsランナーDockerイメージがUbuntu 24.04に更新
- ActionsランナーDockerイメージは、2026年1月を予定しているv2.331.0リリースで、Ubuntu 22.04からUbuntu 24.04に更新されます。このアップデートにより、新しいシステムライブラリ、セキュリティパッチ、最新のツールとの互換性が向上します。actionsランナーに基づくカスタムDockerイメージを使用している場合は、依存関係を確認してUbuntu 24.04との互換性を確保してください。詳細については、プルリクエスト#4018を参照してください。

### [セルフホストランナーの必須アップグレード](#required-upgrade-for-self-hosted-runners)


GitHub Actionsは、CI/CDおよび開発者自動化のための信頼性、スケーラビリティ、新機能の向上を提供するために再アーキテクチャを進めています。この取り組みの一環として、古いバージョンのactionsランナーはサポートされなくなります。


**変更内容**


GitHubは、構成中にセルフホストランナーがv2\.329\.0以降を使用することを要求します。レガシーバージョンを使用しているランナーは、GitHub Actionsへの接続がブロックされます。この変更により、新しいアーキテクチャとの互換性が確保され、将来のロールアウト中にワークフローの失敗が防止されます。


以前は、古いランナーは構成後に自動アップグレードできました。今後、ランナーは登録前に最小バージョン要件を満たす必要があります。


2026年2月9日にブラウンアウトを開始し、完全な施行は2026年3月2日に開始されます。


**必要なアクション**


セルフホストランナーをできるだけ早くv2\.329\.0以降にアップグレードしてください。ガイダンスについては、[セルフホストランナーのドキュメント](https://docs.github.com/actions/reference/runners/self-hosted-runners#runner-software-updates-on-self-hosted-runners)を参照してください。


### [ActionsランナーDockerイメージがUbuntu 24\.04に更新](#actions-runner-docker-image-updated-to-ubuntu-24-04)


ActionsランナーDockerイメージは、2026年1月を予定しているv2\.331\.0リリースで、Ubuntu 22\.04からUbuntu 24\.04に更新されます。このアップデートにより、新しいシステムライブラリ、セキュリティパッチ、最新のツールとの互換性が向上します。actionsランナーに基づくカスタムDockerイメージを使用している場合は、依存関係を確認してUbuntu 24\.04との互換性を確保してください。詳細については、[プルリクエスト\#4018](https://github.com/actions/runner/pull/4018)を参照してください。
