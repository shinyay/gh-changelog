# Gemini 3 ProがVisual Studio、JetBrains IDE、Xcode、Eclipseで利用可能に

## Overview
Gemini 3 Proは、Copilot Enterprise、Copilot Business、Copilot Pro、Copilot Pro+でパブリックプレビューとして利用可能です。

## Detailed Explanation
### Overview
- Gemini 3 Proは、Copilot Enterprise、Copilot Business、Copilot Pro、Copilot Pro+でパブリックプレビューとして利用可能です。
- github.comのGitHub Copilot Chat、GitHub Mobile、Visual Studio Code、Visual Studio、JetBrains IDE、Xcode、Eclipseのチャットモデルピッカー（agent、ask、editモード）からモデルにアクセスできるようになります。まだアクセスできない場合は、後でもう一度確認してください。

### GitHub Copilotでの利用可能性
- この新しいモデルは、以下のユーザーが利用できます：
- Visual Studioバージョン17.14.12以降、18.1.0以降、すべてのモード：ask、agent。
- GitHub Copilot for JetBrainsバージョン1.5.62以降、すべてのモード：ask、edit、agent。
- GitHub Copilot for Xcodeバージョン0.46.0以降、すべてのモード：ask、agent。
- GitHub Copilot for Eclipseバージョン0.14.0以降、すべてのモード：ask、agent。
- 編集者注（2026年1月7日）：Visual Studioがサポートするモードに合わせて、Visual Studioで利用可能なモードを更新しました。

### アクセスの有効化
- Copilot BusinessおよびEnterprise管理者は、Copilot設定でGemini 3 Proポリシーを有効にしてオプトインする必要があります。有効にすると、その組織のユーザーはモデルピッカーにGemini 3 Proが表示されます。
- Copilotで利用可能なモデルの詳細については、モデルに関するドキュメントを参照して、今すぐCopilotを始めましょう。
