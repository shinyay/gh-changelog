# Dependabotセキュリティアップデートがuvをサポート

## Overview
Dependabotは、uvのセキュリティアラートと更新をサポートするようになりました。uv依存関係で脆弱性が検出されると、Dependabotは自動的にセキュリティアラートとプルリクエストを開いて、安全なバージョンに更新できます。
