# GitHub Mobile全体でエージェントセッション作成へのより直接的なアクセス

## Overview
GitHub Mobileで「New Agent Session」オプションの利用可能性を拡大しました！以前は、「Agents」、「Home」、またはリポジトリビューからのみ新しいエージェントセッションを開始できました。

## Detailed Explanation
### Overview
- GitHub Mobileで「New Agent Session」オプションの利用可能性を拡大しました！以前は、「Agents」、「Home」、またはリポジトリビューからのみ新しいエージェントセッションを開始できました。
- 今回のアップデートにより、アプリ内の複数の追加タッチポイントからエージェントセッションを起動できるようになりました。以下が含まれます：
- Actionsレポート/ジョブビュー
- コミットファイルビュー
- プルリクエストメインビュー
- ファイル変更ビュー
- Copilot Chatメッセージオーバーフローメニュー
- ワークフローを効率化し、より迅速なタスクセットアップを促進するために、関連する共有URLまたはテキストメッセージをエージェントセッションコンポーザーに自動的に事前入力します。この機能強化により、アプリ内でいつでもどこでも必要なときにエージェント機能に直接アクセスできるようになります。
- この新しいエクスペリエンスは、iOSおよびAndroid向けGitHub Mobileの最新本番ビルドで利用可能です。
- GitHub Communityでディスカッションに参加してください。

## Impact / Who’s Affected
- 以前は、「Agents」、「Home」、またはリポジトリビューからのみ新しいエージェントセッションを開始できました。
- この新しいエクスペリエンスは、iOSおよびAndroid向けGitHub Mobileの最新本番ビルドで利用可能です。
