# GitHub Actionsの価格設定の更新

## Overview
皆様の投稿を読み、フィードバックをお聞きしました。

## Detailed Explanation
### Update:
- 皆様の投稿を読み、フィードバックをお聞きしました。
- セルフホストGitHub Actionsに関する発表された課金変更を延期し、アプローチを再評価する時間を取ることにしました。
- 2026年1月1日にホストランナーの価格を最大39%引き下げることは継続します。
- Actionsコントロールプレーンの運用には実際のコストが発生しています。また、特に複雑なエンタープライズシナリオにおいて、顧客環境で大規模に機能するようにセルフホストランナーへの投資も行っています。このコンテキストは重要ですが、より多くの方々を計画に含めなかったことで、今回の変更は的を外しました。
- GitHub Actionsを改善する必要があります。まず、開発者、顧客、パートナーとより緊密に会い、耳を傾ける時間を取ります。また、より直接的なフィードバックを収集するためのディスカッションも開設し、そのフィードバックをGitHub Actionsのロードマップに反映させます。GitHub Actionsとプラットフォーム全体での一貫した提供を通じて、皆様の信頼を得られるよう懸命に取り組んでいます。

### Below is the original announcement from December 16, 2025:
- 2026年1月1日、GitHubは使用されるマシンタイプに応じて、GitHubホストランナーの価格を最大39%引き下げます。無料使用分の割り当ては同じままです。
- 2026年3月1日、GitHubはセルフホストランナーの使用に適用される新しい1分あたり$0.002のGitHub Actionsクラウドプラットフォーム料金を導入します。この料金の対象となる使用量は、GitHub Actionsの課金ドキュメントで説明されているように、プランに含まれる分数にカウントされます。
- パブリックリポジトリでのランナー使用は無料のままです。GitHub Enterprise Serverの顧客向けの価格構造に変更はありません。

### Deeper investment in the Actions self-hosted experience
- セルフホスト体験への投資を増やし、Linuxコンテナだけでなく、シナリオのオートスケーリングを提供できるようにします。これには、スケーリングへの新しいアプローチ、新しいプラットフォームサポート、Windowsサポートなど、今後12か月間にわたるものが含まれます。
- Actionsに対して行っている製品投資の詳細については、Executive Insightsページをご覧ください。

### FAQ
- 自分のハードウェアを使用するのになぜ課金されるのですか？歴史的に、セルフホストランナーの顧客は、GitHub Actionsのインフラストラクチャとサービスの多くを無料で利用できました。これは、これらの重要なサービスの維持と進化のコストが、GitHubホストランナーに設定された価格によって主に補助されていたことを意味します。価格設定を更新することで、使用状況とすべてのActionsユーザーに提供される価値に、コストをより密接に調整し、プラットフォーム全体でのさらなるイノベーションと投資を促進します。大多数のユーザー、特に個人や小規模チームは、価格の上昇を見ることはありません。
- Actions価格計算機を使用して、将来のコストを見積もることができます。
- 新しいGitHubホストランナーのレートは何ですか？2026年1月1日に発効する更新されたレートについては、GitHub Actionsランナー価格リファレンスをご覧ください。リストされているレートには、新しい1分あたり$0.002のActionsクラウドプラットフォーム料金が含まれています。
- なぜクラウド上のセルフホストランナーに.002/分が適正価格なのですか？ユーザーによって最も公正で正確と見なされ、市場の他のセルフホストCIソリューションと比較して、1分あたりが決定されました。これは、軽度にも重度にもアクティブな顧客に深刻な影響を与えず、最高のエンドユーザーエクスペリエンスのために高速で柔軟なワークロードを提供する持続可能なオプションだと考えています。
- この価格変更の影響を受けるGitHub Actionsのジョブ実行シナリオはどれですか？
- プライベートリポジトリで実行され、標準GitHubホストまたはセルフホストランナーを使用するジョブ
- より大きなGitHubホストランナーで実行されるすべてのジョブ
- パブリックリポジトリでの標準GitHubホストまたはセルフホストランナーの使用は無料のままです。GitHub Enterprise Serverの価格設定はこの変更の影響を受けません。
- この価格変更はいつ発効しますか？
- GitHubホストランナーの価格引き下げは2026年1月1日に発効します。セルフホストランナーの新しい料金は2026年3月1日から適用されます。価格変更はこれらの日付にすべての顧客に影響します。
- プランで利用可能な無料使用割り当ては変更されますか？2026年3月1日から、セルフホストランナーは無料使用割り当てに含まれ、Linux、Windows、MacOS標準ランナーが今日動作するのと同じように、リスト価格に基づいて利用可能な使用量を消費します。
- セルフホストランナーの使用は無料使用分から消費されますか？はい、課金対象のセルフホストランナーの使用は、プランに関連付けられた無料割り当てから分数を消費できます。
- この価格変更はGitHub Enterprise Serverの顧客にどのように影響しますか？
- この価格変更は、GitHub Enterprise Serverを使用している顧客には影響しません。GitHub Enterprise ServerのセルフホストランナーでActionsジョブを実行している顧客は、無料で実装とともにActionsをホスト、管理、トラブルシューティング、使用し続けることができます。
- プライベートリポジトリでのセルフホストランナーの使用をAzure経由で請求できますか？
- はい、GitHub EnterpriseまたはOrganizationに関連付けられたアクティブなAzureサブスクリプションIDがある限り。
- この変更のGitHub顧客への全体的な影響は何ですか？
- 顧客の96%は請求に変更がありません。この変更の影響を受けるActionsユーザーの4%のうち、このコホートの85%はActions請求が減少し、影響を受ける残りの15%は全体で中央値約$13の増加に直面します。
- GitHubは、エンタープライズ規模の顧客だけでなく、個々の開発者への影響も検討しましたか？個人ユーザー（無料およびProプラン）のうち、先月プライベートリポジトリでGitHub Actionsを使用した人のうち、価格上昇となるのはわずか0.09%で、中央値で月$2未満の増加です。この影響は、これらのユーザーが今日プランに含まれる分数を使用した後のものであり、33時間以上の含まれるGitHubコンピューティングが提供され、パブリックリポジトリの無料使用には影響しません。このユーザーベース全体のさらに2.8%は、これらの変更の結果として月額コストの減少を見るでしょう。残りはこの変更の影響を受けません。
- Actionsの新しい月額コストをどのように計算できますか？
- GitHub Actionsは現在および前年の詳細な使用レポートを提供します。この過去の使用状況と、1月と3月に導入されるレート変更を使用して、新しい価格構造下でのコストを見積もることができます。完全な使用レポートを活用して、価格更新後の予想コストを計算するのに役立つPythonスクリプトを作成しました。
- また、Actions価格計算機を更新し、特に履歴使用量が限られているか、予想される将来の使用量を代表していない場合に、将来のコストを見積もりやすくしました。
