# GitHub DocsでのダイナミックなCopilotプロンプト

## Overview
GitHub Docsの選択されたページから直接GitHub Copilotプロンプトを起動できるようになりました。

## Detailed Explanation
### Overview
- GitHub Docsの選択されたページから直接GitHub Copilotプロンプトを起動できるようになりました。
- ダイナミックなGitHub Copilotプロンプトを導入し、ドキュメントからCopilotにシームレスに移動して、すぐに作業に戻れるようにしました。プロンプトを手動でコピー＆ペーストする代わりに、docsから事前入力されたプロンプトをGitHub Copilotで起動できるようになりました。
- ダイナミックなCopilotプロンプトは3つのタイプで利用可能です：
- インラインプロンプト：GitHub Copilotに一般的な質問を送信
- プロンプトブロック：GitHub CopilotでGitHub関連のプロンプトを開く
- コンテキスト付きプロンプトブロック：GitHub Copilotでプロンプトと関連するコードブロックを起動
- 新しいダイナミックなGitHub Copilotプロンプトを試すには、コードの読みやすさを改善するチュートリアルをチェックしてください。

## Article Content (cleaned)
[GitHub Docs](https://docs.github.com)の選択されたページから直接GitHub Copilotプロンプトを起動できるようになりました。


ダイナミックなGitHub Copilotプロンプトを導入し、ドキュメントから[Copilot](https://github.com/copilot)にシームレスに移動して、すぐに作業に戻れるようにしました。プロンプトを手動でコピー＆ペーストする代わりに、docsから事前入力されたプロンプトをGitHub Copilotで起動できるようになりました。


ダイナミックなCopilotプロンプトは3つのタイプで利用可能です：


* **インラインプロンプト**：GitHub Copilotに一般的な質問を送信
* **プロンプトブロック**：GitHub Copilotでは GitHub\-関連のプロンプトを開く
* **コンテキスト付きプロンプトブロック**：GitHub Copilotでプロンプトと関連するコードブロックを起動


新しいダイナミックなGitHub Copilotプロンプトを試すには、[コードの読みやすさを改善するチュートリアル](https://docs.github.com/copilot/tutorials/copilot-chat-cookbook/refactor-code/improve-code-readability#example-prompt)をチェックしてください。
