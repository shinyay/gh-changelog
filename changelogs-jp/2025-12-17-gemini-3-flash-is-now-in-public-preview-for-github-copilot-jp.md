# Gemini 3 FlashがGitHub Copilotでパブリックプレビュー開始

## Overview
スピードが重要なタスクに最適なGemini 3 Flashが、GitHub Copilotでパブリックプレビューとしてロールアウトされています。

## Detailed Explanation
### Overview
- スピードが重要なタスクに最適なGemini 3 Flashが、GitHub Copilotでパブリックプレビューとしてロールアウトされています。

### Availability in GitHub Copilot
- Gemini 3 Flashは現在、Copilot Pro、Pro+、Business、Enterpriseでロールアウトされています。
- 次の場所でCopilotモデルピッカーからモデルを選択できます：
- Visual Studio Codeのすべてのモード：chat、ask、edit、agent
- github.comのCopilot Chat
- GitHub Mobile（iOSおよびAndroid）
- ロールアウトは段階的です。新しいモデルがまだ表示されない場合は、後ほど再度確認してください。

### Enabling access
- EnterpriseおよびBusinessプラン：管理者はCopilot設定でGemini 3 Flashを有効にする必要があります。有効にすると、その組織のユーザーはCopilotモデルピッカーでモデルを表示できます。
- ProおよびPro+プラン：ユーザーはモデルピッカーでモデルを選択し、1回限りのプロンプトを確認することで、モデルを有効にできます。
- Bring your own key：Visual Studio Code内で、ピッカーからManage Modelsを選択し、Gemini 3 Flashを選択して、プロンプトが表示されたらAPIキーを入力します。

### Learn more
- GitHub Copilotで利用可能なすべてのモデルを探索するには、モデルに関するドキュメントを参照し、Copilotを開始してください。
