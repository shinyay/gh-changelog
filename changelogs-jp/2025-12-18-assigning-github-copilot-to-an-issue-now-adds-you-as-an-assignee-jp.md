# GitHub Copilotをイシューに割り当てると自分自身が担当者として追加される

## Overview
イシューをCopilotに割り当てると、自分自身も自動的に担当者として追加されるようになりました。これにより、`assignee:@me`などのフィルターを使用して作業を追跡しやすくなり、誰がイシューに取り組んでいるかを理解しやすくなります。
