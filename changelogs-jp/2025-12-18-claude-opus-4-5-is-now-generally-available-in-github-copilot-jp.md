# Claude Opus 4.5がGitHub Copilotで一般提供開始

## Overview
Claude Opus 4.5がCopilot Enterprise、Copilot Business、Copilot Pro、Copilot Pro+で一般提供されます。

## Detailed Explanation
### Overview
- Claude Opus 4.5がCopilot Enterprise、Copilot Business、Copilot Pro、Copilot Pro+で一般提供されます。
- github.com、GitHub Mobile、Visual Studio Code、Visual Studio、JetBrains IDE、Xcode、EclipseのGitHub Copilot Chatで、チャットモデルピッカー（agent、ask、editモード）を通じてモデルにアクセスできるようになりました。まだアクセスできない場合は、後ほど再度確認してください。

### Enabling access
- Copilot BusinessおよびCopilot Enterprise管理者は、Copilot設定で新しいClaude Opus 4.5ポリシーを有効にしてオプトインする必要があります。有効にすると、その組織のユーザーは、サポートされているすべてのIDEのモデルピッカーでモデルを表示できます。
- 個人プランの場合、サポートされている任意のIDEでモデルピッカーでモデルを選択し、1回限りのプロンプトを確認することで、モデルを有効にできます。

- フィードバックを共有するには、GitHub Communityに参加してください。
