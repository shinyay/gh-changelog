# データレジデンシー付きGitHub Enterprise CloudでCopilotコードレビュープレビュー機能をサポート

## Overview
データレジデンシー付きGitHub Enterprise Cloudは、Copilotコードレビュープレビュー機能をサポートするようになり、プルリクエストで最新のエージェンティックなコードレビュー機能を試す機会を提供します。

## Detailed Explanation
### Overview
- データレジデンシー付きGitHub Enterprise Cloudは、Copilotコードレビュープレビュー機能をサポートするようになり、プルリクエストで最新のエージェンティックなコードレビュー機能を試す機会を提供します。
- これらのエージェンティック機能はGitHub Universeでパブリックプレビューとして開始され、ユーザーに以下を提供します：
- コード、ディレクトリ構造、参照を含む完全なプロジェクトコンテキストを積極的に収集する豊富なエージェンティックツール呼び出しへのアクセス。
- Copilotコードレビューと一緒に実行するCodeQL、ESLintなどの決定論的ツールを追加および設定する機能。
- Copilotコードレビューで見つかった問題を迅速に解決し、フィードバック間の競合をシームレスに解決するために、Copilotが残したレビュー内からCopilotコーディングエージェントへの簡単な引き継ぎ。
- Copilotエージェントのミッションコントロールを介したCopilotコードレビューエージェントのセッションへの可視性。

データレジデンシー付きGitHub Enterprise Cloudは、[Copilotコードレビュープレビュー機能](https://docs.github.com/copilot/concepts/agents/code-review#new-tools-public-preview-in-github-copilot-code-review)をサポートするようになり、プルリクエストで最新のエージェンティックなコードレビュー機能を試す機会を提供します。


これらのエージェンティック機能は[GitHub Universeでパブリックプレビュー](https://github.blog/changelog/2025-10-28-new-public-preview-features-in-copilot-code-review-ai-reviews-that-see-the-full-picture/)として開始され、ユーザーに以下を提供します：


* コード、ディレクトリ構造、参照を含む完全なプロジェクトコンテキストを積極的に収集する豊富なエージェンティックツール呼び出しへのアクセス。
* Copilotコードレビューと一緒に実行するCodeQL、ESLintなどの決定論的ツールを追加および設定する機能。
* Copilotコードレビューで見つかった問題を迅速に解決し、フィードバック間の競合をシームレスに解決するために、Copilotが残したレビュー内からCopilotコーディングエージェントへの簡単な引き継ぎ。
* Copilotエージェントのミッションコントロールを介したCopilotコードレビューエージェントのセッションへの可視性。


これらの各機能の詳細と有効化方法については、[Copilotドキュメント](https://docs.github.com/copilot/concepts/agents/code-review#new-tools-public-preview-in-github-copilot-code-review)をご覧ください。
