# エンタープライズレベルのプルリクエストアクティビティがCopilot使用メトリクスにパブリックプレビューで含まれる

## Overview
エンタープライズのプルリクエストレベルのアクティビティメトリクス（全体的なプルリクエストの作成とレビューアクティビティを含む）と、GitHub Copilotがそのワークフローにどのように参加しているかを表示できるようになりました。これらはすべて、Enterprise Copilot使用メトリクスAPIレポートで本日利用可能です。

## Detailed Explanation
### Overview
- エンタープライズのプルリクエストレベルのアクティビティメトリクス（全体的なプルリクエストの作成とレビューアクティビティを含む）と、GitHub Copilotがそのワークフローにどのように参加しているかを表示できるようになりました。これらはすべて、Enterprise Copilot使用メトリクスAPIレポートで本日利用可能です。
- 新しいメトリクスには、以下の日次値が含まれます：
- 作成されたプルリクエストの合計
- レビューされたプルリクエストの合計
- Copilotによって作成されたプルリクエストの合計
- Copilotによってレビューされたプルリクエストの合計
- これらのメトリクスはAPIでのみ利用可能で、エンタープライズレベルにスコープされています。実際のプルリクエスト量のコンテキストでCopilotの導入を理解するのに役立つ日次アクティビティシグナルを提供します。
- これらのメトリクスを表示するには、Copilot使用メトリクスポリシーを有効にする必要があります。有効にするには、Enterprisesページに移動し、エンタープライズを選択して、AI Controlsタブをクリックします。左サイドバーでCopilotを選択し、Metricsまでスクロールします。
- エンタープライズオーナー、課金マネージャー、およびView Enterprise Copilot Metrics権限を持つエンタープライズカスタムロールを持つユーザーがAPIにアクセスできます。
- 詳細については、Copilot使用メトリクスに関するドキュメントをご覧ください。
- これらのメトリクスはプレビュー中であり、今後も進化し続けます。組織および個人レベルのメトリクスと、より深いプルリクエストインサイトが次に登場します。
- GitHub Communityでディスカッションに参加してください。

## Impact / Who's Affected
- 新しいメトリクスには、以下の日次値が含まれます：作成されたプルリクエストの合計、レビューされたプルリクエストの合計、Copilotによって作成されたプルリクエストの合計、Copilotによってレビューされたプルリクエストの合計。これらのメトリクスはAPIでのみ利用可能で、エンタープライズレベルにスコープされています。

* 作成されたプルリクエストの合計

* Copilotによって作成されたプルリクエストの合計

これらのメトリクスは**APIでのみ利用可能**で、エンタープライズレベルにスコープされています。実際のプルリクエスト量のコンテキストでCopilotの導入を理解するのに役立つ日次アクティビティシグナルを提供します。


これらのメトリクスを表示するには、**Copilot使用メトリクス**ポリシーを有効にする必要があります。有効にするには、[Enterprises](https://github.com/settings/enterprises)ページに移動し、エンタープライズを選択して、**AI Controls**タブをクリックします。左サイドバーで**Copilot**を選択し、**Metrics**までスクロールします。

詳細については、[Copilot使用メトリクスに関するドキュメント](https://docs.github.com/rest/copilot/copilot-usage-metrics)をご覧ください。


これらのメトリクスはプレビュー中であり、今後も進化し続けます。組織および個人レベルのメトリクスと、より深いプルリクエストインサイトが次に登場します。
