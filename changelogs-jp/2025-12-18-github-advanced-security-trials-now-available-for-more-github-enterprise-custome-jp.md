# GitHub Advanced Securityトライアルがより多くのGitHub Enterprise顧客に利用可能に

## Overview
より多くのGitHub Enterprise顧客が、GitHub Code SecurityおよびGitHub Secret Protectionを評価するために、セルフサービスのGitHub Advanced Securityトライアルを開始できるようになりました。以前にGitHub Advanced Securityトライアルを完了し、Advanced Securityを購入しなかったエンタープライズは、180日後に再度トライアルの資格があります。
