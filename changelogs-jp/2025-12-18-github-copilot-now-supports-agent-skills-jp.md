# GitHub CopilotがAgent Skillsをサポート

## Overview
Copilotに特定の専門的なタスクを、具体的で再現可能な方法で実行する方法を教えるためのAgent Skillsを作成できるようになりました。

## Detailed Explanation
### Overview
- Copilotに特定の専門的なタスクを、具体的で再現可能な方法で実行する方法を教えるためのAgent Skillsを作成できるようになりました。
- Agent Skillsは、プロンプトに関連する場合にCopilotが自動的に読み込む指示、スクリプト、リソースを含むフォルダです。
- Copilotコーディングエージェント、Copilot CLI、Visual Studio Code Insidersのエージェントモードで機能します。Skillsのサポートは1月初旬にVS Codeの安定版に追加される予定です。
- Copilotがスキルがタスクに関連していると判断すると、指示を読み込み、スキルディレクトリに含めたリソースも含めてそれに従います。
- 独自のスキルを作成することも、`anthropics/skills`リポジトリやGitHubのコミュニティが作成した`github/awesome-copilot`コレクションなど、他の人が共有しているスキルを使用することもできます。
- リポジトリの`.claude/skills`ディレクトリにClaude Code用のスキルを既に設定している場合、Copilotは自動的にそれらを認識します。
- 📚 Agent Skillsについて詳しく学ぶ GitHub Communityでディスカッションに参加する。

## Article Content (cleaned)
**Agent Skills**を作成して、Copilotに特定の専門的なタスクを、具体的で再現可能な方法で実行する方法を教えることができるようになりました。


Agent Skillsは、プロンプトに関連する場合にCopilotが自動的に読み込む指示、スクリプト、リソースを含むフォルダです。


Copilotコーディングエージェント、Copilot CLI、[Visual Studio Code Insiders](https://code.visualstudio.com/insiders/)のエージェントモードで機能します。Skillsのサポートは1月初旬にVS Codeの安定版に追加される予定です。


Copilotがスキルがタスクに関連していると判断すると、指示を読み込み、スキルディレクトリに含めたリソースも含めてそれに従います。


独自のスキルを作成することも、[`anthropics/skills`](https://github.com/anthropics/skills)リポジトリやGitHubのコミュニティが作成した[`github/awesome-copilot`](https://github.com/github/awesome-copilot)コレクションなど、他の人が共有しているスキルを使用することもできます。


リポジトリの`.claude/skills`ディレクトリにClaude Code用のスキルを既に設定している場合、Copilotは自動的にそれらを認識します。
