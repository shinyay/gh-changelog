# 日本におけるGitHub Enterprise Cloudデータレジデンシーが一般提供開始

## Overview
日本におけるデータレジデンシーを備えたGitHub Enterprise Cloudが一般提供開始となり、GitHub Enterprise Cloudのお客様がコードとリポジトリデータを保存する場所をより柔軟に選択できるようになりました。これにより、エンタープライズのお客様はチーム向けの単一のグローバルなDevOpsプラットフォームを利用しながら、コードを身近な場所に保管する機能を提供できます。

## Detailed Explanation
### Overview
- 日本におけるデータレジデンシーを備えたGitHub Enterprise Cloudが一般提供開始となり、GitHub Enterprise Cloudのお客様がコードとリポジトリデータを保存する場所をより柔軟に選択できるようになりました。これにより、エンタープライズのお客様はチーム向けの単一のグローバルなDevOpsプラットフォームを利用しながら、コードを身近な場所に保管する機能を提供できます。
- このリリースにより、チームは生産性とコラボレーションを向上させながら、データレジデンシーの設定に合わせてコードをより適切に管理できるようになります。これにより、日本のお客様、または地域内で特定の要件を持つお客様が、コンプライアンス、規制、またはデータレジデンシーのニーズを満たすのに役立ちます。

### What is GitHub Enterprise Cloud with data residency?
- GitHub Enterprise Cloudは、Microsoft Azureを基盤とするGitHub Enterpriseのマルチテナント型エンタープライズSaaS展開オプションです。GitHubエコシステムと統合されたツールと機能のスイートを備えた、単一のエンドツーエンドのDevOpsプラットフォームを提供し、組織が大規模にソフトウェアを構築、保護、出荷できるようにします。
- 日本でのデータレジデンシーの導入により、お客様はコードを保存する場所を柔軟に選択できるようになりました。データレジデンシーを備えたGitHub Enterprise Cloudは、Microsoft Azureのグローバルに分散されたデータセンターインフラストラクチャを基盤としており、チームがどこにいても高可用性、セキュリティ、パフォーマンスを提供します。

### Who is this available for?
- 日本におけるデータレジデンシーを備えたGitHub Enterprise Cloudは、コードとリポジトリデータをその地域に保管する必要があるお客様が利用できます。追加地域のサポートは現在計画中であり、今後の変更ログ更新で発表される予定です。

### How can you access GitHub Enterprise Cloud with data residency in Japan?
- 今すぐ営業チームに連絡するか、データレジデンシーを備えたGitHub Enterprise Cloudの30日間無料トライアルにサインアップして開始できます。また、Docsにアクセスし、GitHub Enterprise Cloudの展開オプションを探索して詳細を学ぶこともできます。
