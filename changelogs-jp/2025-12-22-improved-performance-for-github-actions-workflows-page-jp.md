# GitHub Actionsワークフローページのパフォーマンス向上

## Overview
GitHub Actionsワークフローページで、300以上のジョブを持つワークフローを正常にレンダリングできるようになりました。大規模なワークフローをスムーズに処理するために、遅延読み込みを実装しました。

## Detailed Explanation
### Overview
- GitHub Actionsワークフローページで、300以上のジョブを持つワークフローを正常にレンダリングできるようになりました。大規模なワークフローをスムーズに処理するために、遅延読み込みを実装しました。
- さらに、ステータスに基づいてジョブをフィルタリングできるようになりました。つまり、ワークフローページから失敗したジョブまたは進行中のジョブのみを表示できます。
- これらの改善により、広範なジョブマトリックスを持つ複雑なワークフローや大規模なモノレポのセットアップを実行している場合でも、チームがワークフロー実行を表示できるようになります。

## Impact / Who’s Affected
- つまり、ワークフローページから失敗したジョブまたは進行中のジョブのみを表示できます。
