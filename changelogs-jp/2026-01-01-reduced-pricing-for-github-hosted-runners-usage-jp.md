# GitHub-hosted runnersの使用料金が削減

## Overview
GitHub-hosted runnersの削減されたコストが現在有効になっています。この更新は2025年12月16日に発表されました。GitHubユーザーは、使用するマシンに応じて、GitHub-hosted runnersの料金が最大39%削減されることになります。
