# Gemini 3 FlashがVisual Studio、JetBrains IDE、Xcode、Eclipseで利用可能に

## Overview
Gemini 3 Flashが、Copilot Enterprise、Copilot Business、Copilot Pro、Copilot Pro+向けにパブリックプレビューで利用可能になりました。

## Detailed Explanation
### Overview
- Gemini 3 Flashが、Copilot Enterprise、Copilot Business、Copilot Pro、Copilot Pro+向けにパブリックプレビューで利用可能になりました。
- github.com、GitHub Mobile、Visual Studio Code、Visual Studio、JetBrains IDE、Xcode、EclipseのGitHub Copilot Chatで、チャットモデルピッカー（agent、ask、editモード）を通じてモデルにアクセスできるようになります。まだアクセスできない場合は、後ほど再度ご確認ください。

### Availability in GitHub Copilot
- この新しいモデルは、以下のユーザーが利用できます：
- Visual Studioバージョン17.14.12以降、18.1.0以降、すべてのモード：ask、agent。
- GitHub Copilot for JetBrainsバージョン1.5.62以降、すべてのモード：ask、edit、agent。
- GitHub Copilot for Xcodeバージョン0.46.0以降、すべてのモード：ask、agent。
- GitHub Copilot for Eclipseバージョン0.14.0以降、すべてのモード：ask、agent。
- 編集者注（2026年1月7日）：Visual Studioがサポートするモードに合わせて、利用可能なモードを更新しました。

### Enabling access
- Copilot BusinessおよびEnterpriseの管理者は、Copilot設定でGemini 3 Flashポリシーを有効にすることでオプトインする必要があります。有効にすると、その組織のユーザーはモデルピッカーにGemini 3 Flashが表示されます。
- Copilotで利用可能なモデルの詳細については、モデルに関するドキュメントを参照し、今すぐCopilotを始めましょう。
