# Claude、Google、OpenAIの一部GitHub Copilotモデルの廃止予定

## Overview
高速で高品質なCopilotエクスペリエンスへのコミットメントの一環として、古いモデルを定期的に評価し、より新しく、より高性能なモデルを優先して廃止しています。

## Detailed Explanation
### Overview
- 高速で高品質なCopilotエクスペリエンスへのコミットメントの一環として、古いモデルを定期的に評価し、より新しく、より高性能なモデルを優先して廃止しています。
- 2026年2月17日に、すべてのGitHub Copilotエクスペリエンス（Copilot Chat、インライン編集、askおよびagentモード、コード補完を含む）で次のモデルを廃止します：
- これらの日付より前にサポートされているモデルを使用するようにワークフローと統合を更新してください。Copilot Enterprise管理者は、Copilot設定でモデルポリシーを通じて代替モデルへのアクセスを有効にする必要がある場合があります。管理者として、個々のCopilot設定を確認し、特定のモデルに対してポリシーが有効になっていることを確認することで、可用性を確認できます。有効にすると、VS Codeおよびgithub.comのCopilot Chatモデルセレクターにモデルが表示されます。モデルが廃止されたら、削除するためのアクションは必要ありません。
- 質問や懸念事項があるGitHub Enterpriseのお客様は、アカウントマネージャーに連絡してサポートを受けることをお勧めします。
