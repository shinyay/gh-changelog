# GPT-5.2-Codex が GitHub Copilot で一般提供開始

## Overview
GPT-5.2-Codex が Copilot Enterprise、Copilot Business、Copilot Pro、Copilot Pro+ で一般提供されています。

## Detailed Explanation
### Overview
- GPT-5.2-Codex が Copilot Enterprise、Copilot Business、Copilot Pro、Copilot Pro+ で一般提供されています。

### Availability in GitHub Copilot
- 次のいずれかから Copilot モデルピッカーでモデルを選択できます:
- Visual Studio Code のすべてのモード: chat、ask、edit、agent
- github.com の Copilot Chat
- GitHub Mobile (iOS および Android)
- Copilot コーディングエージェント (Copilot Pro および Pro+ サブスクライバーのみ)
- Copilot CLI
- ロールアウトは段階的です。新しいモデルがまだ表示されない場合は、後ほど再度ご確認ください。

### Enabling access
- Enterprise および Business プラン : 管理者は Copilot 設定で OpenAI GPT-5.2-Codex を有効にする必要があります。有効にすると、その組織のユーザーは Copilot モデルピッカーに対応するモデルが表示されます。
- Pro および Pro+ プラン : ユーザーはモデルピッカーでモデルを選択し、1回限りのプロンプトを確認することでモデルを有効にできます。
- Bring your own key : Visual Studio Code 内で、ピッカーから Manage Models を選択し、OpenAI GPT-5.2-Codex を選択して、プロンプトが表示されたら API キーを入力します。

### Learn more
- GitHub Copilot で利用可能なすべてのモデルを探索するには、モデルに関するドキュメントを参照して、Copilot を始めてください。
