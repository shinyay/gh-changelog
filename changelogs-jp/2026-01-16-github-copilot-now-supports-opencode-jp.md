# GitHub Copilot が OpenCode をサポート

## Overview
GitHub は、Copilot Pro、Pro+、Business、または Enterprise サブスクリプションを OpenCode で使用することを公式にサポートしています。

## Detailed Explanation
### Overview
- GitHub は、Copilot Pro、Pro+、Business、または Enterprise サブスクリプションを OpenCode で使用することを公式にサポートしています。
- GitHub Copilot は正式なパートナーシップを通じて OpenCode での認証を完全にサポートし、既存の Copilot サブスクリプションを開発ワークフローのより多くの部分で使用できるようになりました。

### What is OpenCode?
- OpenCode は、ターミナル、IDE、またはデスクトップでコードを書くのを支援するオープンソースエージェントです。詳細は OpenCode のサイトで確認できます。

### How it works
- OpenCode と GitHub Copilot を始めるのは簡単です:
- OpenCode で `/connect` を実行し、GitHub Copilot を選択します。
- GitHub デバイスログインフローを完了します。
- Copilot を搭載した OpenCode の使用を開始します。

### Who can use this feature
- 有料の GitHub Copilot サブスクリプション (Pro、Pro+、Business、または Enterprise) を持つすべての開発者が、Copilot 認証情報を使用して OpenCode に認証できるようになりました。追加の AI ライセンスは必要ありません。
- OpenCode の Copilot プロバイダー設定の詳細をご確認ください。
- GitHub Community 内でディスカッションに参加してください。
