# GitHub Issuesの読み込みが高速化

## Overview
GitHub Issuesが大幅に高速化され、issueビューの35%が200ms未満で完了するようになりました（年初は2%でした）。

## Detailed Explanation
### 概要
- GitHub Issuesが大幅に高速化され、issueビューの35%が200ms未満で完了するようになりました（年初は2%でした）。
- これは、GitHub IssuesとGitHub全体を即座に感じられるようにするために提供する一連のパフォーマンス改善の第一弾です。

- これらのパフォーマンス改善は、本日github.comで公開されています。これらの恩恵を受けるには、サインインしている必要があります。設定やオプトインは不要です。

- リポジトリインデックス

- GitHub Project
- これらの改善を追加の経路にも展開中です。

これらの改善を追加の経路にも展開中です。
