# 改善されたプルリクエスト「Files changed」ページがデフォルトで有効に

## Overview
現在パブリックプレビュー中の改善されたプルリクエスト「Files Changed」エクスペリエンスが、すべてのユーザーのデフォルトエクスペリエンスとしてロールアウトされています！ 🎉

## Detailed Explanation
### 概要
- 現在パブリックプレビュー中の改善されたプルリクエスト「Files Changed」エクスペリエンスが、すべてのユーザーのデフォルトエクスペリエンスとしてロールアウトされています！ 🎉
- この改善されたエクスペリエンスは、既存のプルリクエストユーザーにとって使い慣れたものになるように設計されていますが、パフォーマンス、アクセシビリティ、ユーザー生産性が向上しています。クラシックエクスペリエンスに戻す必要がある場合、オプトアウトオプションは引き続き利用可能です。

### ✨ 新機能
- クラシックエクスペリエンスから移行する場合、新しいエクスペリエンスは使い慣れたものに感じられますが、いくつかの重要な改善が追加されています：
- 任意の行にコメント：変更されたファイルの任意の行にコメントできるようになりました（変更の周辺の行だけでなく）。
- ページを切り替えずに説明を表示：「Files changed」ページの新しいOverviewパネルからプルリクエストの説明を表示できるようになりました。
- レビューが容易に 強化されたファイルツリー：ファイルツリーはサイズ変更可能になり、コメント、エラー、警告があるファイルを示すインジケータが表示されます。保留中のコメントをレビュー：保留中のコメントがレビュー送信パネルに表示されるようになり、送信前にフィードバックを再確認できます。ドラフトコメント：新しいコメントと返信はローカルに保存されるため、誤ってブラウザを閉じたりページを更新したりしても失われません。
- ページのリロードが減少：Refreshをクリックして新しい変更を取り込む、分割モードと統合モードを切り替える、その他のタスクで、完全なページ更新が不要になりました。
- アクセシビリティと可読性：キーボードナビゲーション、スクリーンリーダーランドマーク、行間隔オプションの増加が利用可能になり、すべての人がエクスペリエンスを利用できるようになりました。
- 強化されたファイルツリー：ファイルツリーはサイズ変更可能になり、コメント、エラー、警告があるファイルを示すインジケータが表示されます。
- 保留中のコメントをレビュー：保留中のコメントがレビュー送信パネルに表示されるようになり、送信前にフィードバックを再確認できます。
- ドラフトコメント：新しいコメントと返信はローカルに保存されるため、誤ってブラウザを閉じたりページを更新したりしても失われません。

### 🚀 大規模プルリクエストの改善
- 新しい「Files changed」エクスペリエンスで、大規模なプルリクエストをレビューするための新しい実験的モードが利用可能になりました。このモードは、仮想化を使用してブラウザが管理する必要があるDOM要素とイベントリスナーの数を減らします。これにより、特に低速なマシンでメモリ使用率とページの応答性が大幅に向上します。また、単一ファイルモードによって引き起こされる多くのユーザビリティ上の課題も回避されます。
- この新しいモードを試すには、大規模なプルリクエストを表示したときに表示されるバナーで「Try a new experimental mode」をクリックします：
- 変更の完全なセットがブラウザによってレンダリングされないため、一部のブラウザ機能が期待どおりに動作しない場合があります：
- diff内容の検索（例：関数名の検索）。
- すべてのテキストの選択（例：他の場所にコピーして処理するため）。
- ページの印刷またはエクスポート。
- ページ上のすべてのdiff内容を必要とするブラウザ拡張機能の使用。
- このモードで単一ファイルモードに戻すには、大規模なプルリクエストを表示したときのバナーで「Switch to single file mode」リンクをクリックします。

### 🔧 修正と改善
- 改善：diffのコメントワークフローがより直感的で信頼性が高くなり、複数行選択、行番号クリック選択、新しいコメントの正しいフォーカス動作、完全に機能するdiffハンクが提供されます。
- 修正：提案された変更が誤って適用される、部分的に適用される、行が予期せず削除される、または変更された行やコンテキスト行で誤って動作する問題。
- 修正：アクティブなレビュー中にユーザーがコメントや古いスレッドを追加、編集、削除、返信、またはジャンプできない複数のバグ。
- 修正：フィルター、スクロール、正しいスレッドへのナビゲーションが期待どおりに機能しないコメントサイドパネルの問題。
- 修正：特にSafariでのコメント返信での入力遅延とインタラクションラグ。
- 修正：PとNを使用したコミット間のナビゲーションのキーボードショートカットが再び確実に機能するようになりました。
- 修正：不正なスクロール位置、スティッキーヘッダーが誤って動作する、表示されたファイル数の不一致、モバイルでの予期しないスクロールジャンプなどの「Files Changed」UIの問題。
- 修正：コメントプレビュー、ツールバー、ドロップダウン、オーバーレイに影響する視覚的およびインタラクションのバグで、一般的なレビューアクションをブロックまたは混乱させる可能性があったもの。
- 修正：スクリーンリーダーのアナウンス、キーボードナビゲーション、ツールチップのラベル付け、フォーカス順序、小さい画面でのコンテンツの可視性に影響するアクセシビリティの問題。

### ♥️ フィードバック
- このエクスペリエンスを改善するために、あなたのフィードバックは不可欠です。問題の報告、質問、既知の問題の確認には、「Files Changed」プレビューフィードバックディスカッションを使用してください。
