# GPT-5.2-CodexがVisual Studio、JetBrains IDE、Xcode、Eclipseで利用可能に

## Overview
GPT-5.2-CodexがCopilot Enterprise、Copilot Business、Copilot Pro、Copilot Pro+で一般提供されています。

## Detailed Explanation
### 概要
- GPT-5.2-CodexがCopilot Enterprise、Copilot Business、Copilot Pro、Copilot Pro+で一般提供されています。
- github.com、GitHub Mobile、Visual Studio Code、Visual Studio、JetBrains IDE、Xcode、EclipseのGitHub Copilot Chatでチャットモデルピッカー（agent、ask、editモード）を通じてモデルにアクセスできるようになりました。まだアクセスできない場合は、後でもう一度確認してください。

### GitHub Copilotでの利用可能性
- この新しいモデルは、以下のユーザーが利用可能になります：
- Visual Studioバージョン17.14.19以降、18.0.0以降、すべてのモード：ask、agent。
- GitHub Copilot for JetBrainsバージョン1.5.61以降、すべてのモード：ask、edit、agent。
- GitHub Copilot for Xcodeバージョン0.45.0以降、すべてのモード：ask、agent。
- GitHub Copilot for Eclipseバージョン0.13.0以降、すべてのモード：ask、agent。

### アクセスの有効化
- Copilot BusinessおよびEnterprise管理者は、Copilot設定でGPT-5.2-Codexポリシーを有効にすることでオプトインする必要があります。有効にすると、その組織のユーザーはモデルピッカーにGPT-5.2-Codexが表示されます。
- Copilotで利用可能なモデルの詳細については、モデルに関するドキュメントを参照し、今すぐCopilotを始めましょう。
