# Visual Studio 向け GitHub Copilot — 2026年1月アップデート

## Overview
2026年1月の更新では、色付き（シンタックスハイライト付き）のコード補完、補完の部分採用、エディタの生産性改善などが追加されたと説明されています。

## Detailed Explanation
### 概要
- 2026年1月の更新では、色付きコード補完、補完の部分採用、生産性改善などが含まれます。

### 主な変更点（Highlights）
- **Colorized code completions**: コード補完にシンタックスハイライトが付くようになり、変数・関数などの要素を見分けやすくなります。設定で **Use colorized text for code completions** を選択して有効化します。
- **Click to partially accept code completions**: 提案の中をクリックして、提案の一部だけを採用できます。ホバーするとセグメントがハイライトされ、カーソル位置までを受け入れられます。
- **Productivity improvements**: スクロール高速化、HTML リッチなコピー&ペースト、構文的な行圧縮など、エディタの操作性改善が含まれます。
- **Streamlined Markdown preview controls**: Split Preview / Open Preview / Edit Markdown の切り替えができ、Mermaid 図向けのズームも改善されます。また Copilot Chat に **Preview** ボタンが追加され、生成した markdown を描画して編集しやすくなります（Stable チャネルで利用可能と記載）。

### 次に向けて
- Visual Studio blog をフォローしてロードマップ更新やフィードバック機会を確認できます。
- リリースノート、動画、コミュニティ等は Visual Studio Hub が案内されています。

## Impact / Who’s Affected
- Visual Studio で Copilot を使う開発者は、補完の可読性向上や部分採用により、提案の取り込みがしやすくなる可能性があります。
- Stable チャネル利用者は、Copilot Chat の Markdown Preview ボタンなどの恩恵を受けます。

## Caveats / Limitations
- 一部機能の提供チャネル（Insiders / Stable）に言及があり、利用可否は環境に依存する可能性があります（詳細は記事内リンク参照）。
