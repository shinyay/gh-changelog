# Web 版 Copilot Chat のツール呼び出し表示などの改善

## Overview
github.com 上の GitHub Copilot Chat 体験を、よりスムーズで高速にするための改善が追加されました。

## Detailed Explanation
### ツール呼び出し（tool call）の表示
- Copilot にメッセージを送信した際、レスポンス内でツール呼び出しの詳細がより明確に表示されるようになります。
- これにより、次のことがしやすくなると説明されています。
  - Copilot がリクエストを満たすために実行しているステップを理解する
  - 何かが失敗したときに方向修正する
  - 各ツール呼び出しで生成された参照情報を確認する

### チャットのエクスポート（JSON / Markdown）
- 会話を後で参照できるよう、チャットをエクスポートできるようになりました。
- 手順:
  - ダウンロードしたいチャットを選択
  - 右上などの「Manage chat」メニュー（more menu）を開き、Export を選択
  - JSON または Markdown を選択

### リポジトリ添付の改善
- チャットにリポジトリを添付する際の検索性が改善され、目的のリポジトリを見つけやすくなります。
- 検索時により良いリポジトリ推薦が表示されるようになります。

### フィードバック
- これらの改善について GitHub Community で意見交換できると案内されています。

## Key Changes
- チャットレスポンス内でツール呼び出しの詳細が表示され、手順や参照の可視化が強化されました。
- チャットを JSON / Markdown としてエクスポートできるようになりました。
- リポジトリ添付の検索/推薦体験が改善されました。
