# 個人ユーザー向け Copilot モデル有効化体験の簡素化

## Overview
Copilot の各機能で最新モデルを見つけて使いやすくするための改善が入りました。Copilot Pro+ / Pro / Free の個人ユーザーは、新しくリリースされたモデルを Copilot 内から直接利用でき、ポリシー管理や設定ページの操作が不要になります。

## Detailed Explanation
### 変更点
- Copilot Pro+ / Pro / Free の個人ユーザーは、新しいモデルを Copilot の機能内からすぐ使えるようになります。
- これまで必要だった「ポリシーの管理」や「設定ページへ移動」が不要になると説明されています。

### この変更によりできること
- 新しいモデルが、追加の操作なしで Copilot 機能内からすぐに利用可能になります。
- モデルは自分で選ぶことも、Auto mode をオンにして GitHub に選ばせることもできます。
- 選択したモデルはすぐに動作します。

### データ利用ポリシー
- 既存のデータ利用ポリシーは変更されず、引き続き適用されると明記されています。
- 追加の説明は Trust Center へのリンクで案内されています。

### ロールアウト
- この変更は「翌日までに」ロールアウトされると記載されています。
- まだ表示されない場合は、しばらくしてから再確認するよう案内されています。

## Impact / Who’s Affected
- Copilot Pro+ / Pro / Free の個人ユーザーは、新しいモデルを設定ページに行かずに利用できるようになります。

## Caveats / Limitations
- ロールアウトは翌日までに段階的に行われるため、反映までタイムラグがある可能性があります。
