# Claude Opus 4.6 の高速モードが GitHub Copilot でプレビュー提供開始

## Overview
Claude Opus 4.6 の高速モードが GitHub Copilot のリサーチプレビューとして展開されています。この高速オプションは、Opus 4.6 と同等の知能を維持しながら、出力トークン速度を最大 2.5 倍に高速化します。本リリースは初期段階の実験的なものであり、大幅なインファレンスの高速化に焦点を当てています。

## Detailed Explanation
### Overview
- Claude Opus 4.6 の高速モードが GitHub Copilot のリサーチプレビューとして展開中です。出力トークン速度を最大 2.5 倍に高速化し、Opus 4.6 と同等の知能を維持します。初期段階の実験的なリリースです。

- Claude Opus 4.6 の高速モードは Copilot Enterprise ユーザーに提供されます。

- Copilot Enterprise プラン管理者が Copilot 設定で「Fast mode for Claude Opus 4.6」ポリシーを有効にする必要があります。
