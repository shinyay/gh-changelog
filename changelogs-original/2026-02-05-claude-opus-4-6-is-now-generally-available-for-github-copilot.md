# Claude Opus 4.6 is now generally available for GitHub Copilot

## Overview
Claude Opus 4.6, Anthropic’s latest model, is now rolling out in GitHub Copilot. In early testing, Claude Opus 4.6 excels in agentic coding, with specialization on especially hard tasks requiring planning and tool calling.

## Detailed Explanation
### Overview
- Claude Opus 4.6, Anthropic’s latest model, is now rolling out in GitHub Copilot. In early testing, Claude Opus 4.6 excels in agentic coding, with specialization on especially hard tasks requiring planning and tool calling.

### Availability in GitHub Copilot
- Claude Opus 4.6 will be available to Copilot Pro, Pro+, Business, and Enterprise users.
- You’ll be able to select the model in the model picker in:
- Visual Studio Code in all modes: chat, ask, edit, agent
- Visual Studio in all modes: agent, ask
- github.com
- GitHub Mobile iOS and Android
- GitHub CLI
- GitHub Copilot coding agent (for Pro and Pro+ accounts only)
- Rollout will be gradual. Check back soon if you don’t see it yet.

### Enabling access
- Copilot Enterprise and Copilot Business plan administrators must enable the Claude Opus 4.6 policy in Copilot settings.

- To explore all models available in GitHub Copilot, see our documentation on models and get started with Copilot.
