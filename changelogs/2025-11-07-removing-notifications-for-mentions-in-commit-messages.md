# Removing notifications for @mentions in commit messages

## Overview
To reduce unwanted noise and help you focus on what matters most, we’re removing notifications generated from mentions in commit messages. Previously, mentioning a user or team in a commit message (e.g., fixes bug for @username) triggered a notification, just like in issues or pull requests.

## Detailed Explanation
### Overview
- To reduce unwanted noise and help you focus on what matters most, we’re removing notifications generated from mentions in commit messages. Previously, mentioning a user or team in a commit message (e.g., fixes bug for @username) triggered a notification, just like in issues or pull requests.
- Editor’s note (November 7, 2025): Updated terminology for the type of notifications.

### What’s changing
- On December 8th, Users and teams will no longer receive notifications when mentioned in a commit message. Mentions in issues, pull requests, and discussions will continue to trigger notifications.
