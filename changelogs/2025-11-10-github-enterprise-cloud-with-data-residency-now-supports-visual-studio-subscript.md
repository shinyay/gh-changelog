# GitHub Enterprise Cloud with data residency now supports Visual Studio subscriptions

## Overview
Visual Studio subscriptions for both subscription and usage-based GitHub Enterprise now support data residency. This allows you to access GitHub Enterprise features through your Visual Studio subscription while maintaining compliance with regional data requirements.

## Detailed Explanation
### Overview
- Visual Studio subscriptions for both subscription and usage-based GitHub Enterprise now support data residency. This allows you to access GitHub Enterprise features through your Visual Studio subscription while maintaining compliance with regional data requirements.
