# Fine-grain permissions for Copilot usage metrics now available

## Overview
Enterprise role: View enterprise Copilot metrics

## Detailed Explanation
### What’s new?
- Enterprise role: View enterprise Copilot metrics
- You can now access the Copilot Usage Metrics APIs and Dashboard if you are assigned the new enterprise role: View enterprise Copilot metrics
- This important expansion means that all users assigned this role , not just enterprise admins or billing managers, are empowered to view granular Copilot usage metrics for your enterprise.

### Who can now access Copilot usage metrics?
- Any user with the View enterprise Copilot metrics role
- Enterprise admins and billing managers continue to have default access.

### How to get started
- Ensure you have the Copilot usage metrics policy enabled under AI Controls > Copilot .
- Assign your users the new View enterprise Copilot metrics enterprise role under People > Enterprise roles .
- Advise your newly assigned users to visit the Copilot Usage Metrics APIs and Dashboard to begin leveraging insights today.
- Learn more about Copilot usage metrics .
