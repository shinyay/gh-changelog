# Custom agents available in GitHub Copilot for JetBrains, Eclipse, and Xcode now in public preview

## Overview
Custom agents are now in public preview for JetBrains IDEs, Eclipse, and Xcode, allowing you to package reusable expertise, guardrails, and tool access into AI agents tailored to your project or organization.

## Detailed Explanation
### Overview
- Custom agents are now in public preview for JetBrains IDEs, Eclipse, and Xcode, allowing you to package reusable expertise, guardrails, and tool access into AI agents tailored to your project or organization.

### What’s new
- Custom agents bring customization to your chat mode by letting you specify name, description, tools, and models. Create specialized AI agents tailored to your workflows and coding standards in JetBrains, Eclipse, and Xcode. Define agents using Markdown files that specify prompts, so you can pick them up and run in your IDEs quickly.

### Try it out
- Update your IDE : Install the latest Copilot extension for your JetBrains IDE, Eclipse, or Xcode to access the custom agents preview.
- Create a custom agent : Follow the custom agents quickstart and how-to guide to generate a starter configuration file describing the agent’s name, description, instructions, knowledge sources, and tool policies.
- Pick the custom agent : In Copilot Chat, select your custom agent from the list of available agents.
- Test the agent : Start the conversation under the current custom agent.
