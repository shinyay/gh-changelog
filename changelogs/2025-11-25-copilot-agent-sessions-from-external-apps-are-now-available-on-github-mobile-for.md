# Copilot agent sessions from external apps are now available on GitHub Mobile for Android

## Overview
We’ve made it easier to jumpstart agent sessions straight from your favorite Android apps. With our new Android sharesheet custom action in GitHub Mobile for Android, you can now share text from external apps and instantly create a Copilot agent session. Anyone using the GitHub Android app can take ideas drafted elsewhere—notes, messages, or plans—and send them right into a new agent session to get Copilot working for you. This update streamlines your workflow and helps you turn inspiration into action faster. Try it out the next time you’re brainstorming a feature or mapping out a project.
