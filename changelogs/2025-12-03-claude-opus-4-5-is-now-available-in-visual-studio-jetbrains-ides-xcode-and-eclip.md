# Claude Opus 4.5 is now available in Visual Studio, JetBrains IDEs, Xcode, and Eclipse

## Overview
Claude Opus 4.5 is available in public preview to Copilot Enterprise, Copilot Business, Copilot Pro, and Copilot Pro+.

## Detailed Explanation
### Overview
- Claude Opus 4.5 is available in public preview to Copilot Enterprise, Copilot Business, Copilot Pro, and Copilot Pro+.
- You’ll now be able to access the model in GitHub Copilot Chat on github.com, GitHub Mobile, Visual Studio Code, Visual Studio, JetBrains IDEs, Xcode, and Eclipse through the chat model picker (agent, ask, and edit modes). If you do not yet have access, please check again later.

### Enabling access
- Copilot Business and Enterprise administrators must opt in by enabling the Claude Opus 4.5 policy in Copilot settings. Once enabled, users in that organization will see Claude Opus 4.5 in the model picker.
- To learn more about the models available in Copilot, see our documentation on models and get started with Copilot today.
