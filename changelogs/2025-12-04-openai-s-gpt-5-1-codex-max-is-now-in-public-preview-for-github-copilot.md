# OpenAI’s GPT-5.1-Codex-Max is now in public preview for GitHub Copilot

## Overview
GPT-5.1-Codex-Max is now rolling out in public preview in GitHub Copilot.

## Detailed Explanation
### Overview
- GPT-5.1-Codex-Max is now rolling out in public preview in GitHub Copilot.

### Availability in GitHub Copilot
- OpenAI’s GPT-5.1-Codex-Max will be available to Copilot Pro, Pro+, Business, and Enterprise.
- You’ll be able to select the model in the Copilot Chat model picker from:
- Visual Studio Code in all modes: chat, ask, edit, agent
- Copilot Chat in github.com
- Copilot Chat in GitHub Mobile (iOS and Android)
- Copilot CLI
- Rollout will be gradual. If you don’t see the new models yet, check back soon.

### Enabling access
- Enterprise and Business plans : Administrators must enable the GPT-5.1-Codex-Max policy in Copilot settings. Once enabled, users in that organization will see the respective model in the Copilot Chat model picker.
- Pro and Pro+ plans : Users can enable the model by selecting it in the model picker and confirming the one-time prompt.
- Bring your own key : Select Manage Models from the picker, choose GPT-5.1-Codex-Max, and enter your API key when prompted.
