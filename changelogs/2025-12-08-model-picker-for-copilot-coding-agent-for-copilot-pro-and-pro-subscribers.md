# Model picker for Copilot coding agent for Copilot Pro and Pro+ subscribers

## Overview
Copilot Pro and Pro+ subscribers can now pick a model when using Copilot coding agent , our asynchronous, autonomous background agent.

## Detailed Explanation
### Overview
- Copilot Pro and Pro+ subscribers can now pick a model when using Copilot coding agent , our asynchronous, autonomous background agent.
- When starting a task from the agents tab, agents panel, issue assignment on github.com, or the Raycast extension, you can use the dropdown to pick from the following models:
- Auto
- Anthropic Claude Opus 4.5
- Anthropic Claude Sonnet 4.5
- OpenAI GPT-5.1-Codex-Max
- If you leave the setting on Auto , Copilot optimizes speed and performance by automatically choosing a model based on availability.
- The model picker is now available to Copilot Pro and Pro+ subscribers, and coming soon to Copilot Business and Enterprise subscribers, along with more Copilot coding agent integrations like Visual Studio Code and GitHub Mobile .
- To learn more, head to our documentation about model selection in Copilot coding agent .

## Article Content (cleaned)
Copilot Pro and Pro\+ subscribers can now pick a model when using [Copilot coding agent](https://docs.github.com/copilot/concepts/agents/coding-agent/about-coding-agent), our asynchronous, autonomous background agent.


When starting a task from the agents tab, agents panel, issue assignment on github.com, or the Raycast extension, you can use the dropdown to pick from the following models:


* Auto
* Anthropic Claude Opus 4\.5
* Anthropic Claude Sonnet 4\.5
* OpenAI GPT\-5\.1\-Codex\-Max


If you leave the setting on **Auto**, Copilot optimizes speed and performance by automatically choosing a model based on availability.


The model picker is now available to Copilot Pro and Pro\+ subscribers, and coming soon to Copilot Business and Enterprise subscribers, along with more Copilot coding agent integrations like [Visual Studio Code](https://docs.github.com/enterprise-cloud@latest/copilot/how-tos/use-copilot-agents/coding-agent/create-a-pr#asking-copilot-to-create-a-pull-request-from-copilot-chat-in-visual-studio-code) and [GitHub Mobile](https://docs.github.com/enterprise-cloud@latest/copilot/how-tos/use-copilot-agents/coding-agent/create-a-pr#asking-copilot-to-create-a-pull-request-from-github-mobile).


To learn more, head to [our documentation about model selection in Copilot coding agent](https://docs.github.com/copilot/how-tos/use-copilot-agents/coding-agent/changing-the-ai-model).
