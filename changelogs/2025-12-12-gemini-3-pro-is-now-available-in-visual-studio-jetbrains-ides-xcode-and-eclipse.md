# Gemini 3 Pro is now available in Visual Studio, JetBrains IDEs, Xcode, and Eclipse

## Overview
Gemini 3 Pro is available in public preview to Copilot Enterprise, Copilot Business, Copilot Pro and Copilot Pro+.

## Detailed Explanation
### Overview
- Gemini 3 Pro is available in public preview to Copilot Enterprise, Copilot Business, Copilot Pro and Copilot Pro+.
- You’ll now be able to access the model in GitHub Copilot Chat on github.com, GitHub Mobile, Visual Studio Code, Visual Studio, JetBrains IDEs, Xcode, and Eclipse through the chat model picker (agent, ask, and edit modes). If you do not yet have access, please check again later.

### Availability in GitHub Copilot
- This new model will be available to users in:
- Visual Studio versions 17.14.12 and later, 18.1.0 and later, in all modes: ask, agent.
- GitHub Copilot for JetBrains versions 1.5.62 and later, in all modes: ask, edit, agent.
- GitHub Copilot for Xcode versions 0.46.0 and later, in all modes: ask, agent.
- GitHub Copilot for Eclipse versions 0.14.0 and later, in all modes: ask, agent.
- Editor’s note (January 7, 2026): Updated the available modes for Visual Studio to match the modes Visual Studio supports.

### Enabling access
- Copilot Business and Enterprise administrators must opt in by enabling the Gemini 3 Pro policy in Copilot settings. Once enabled, users in that organization will see Gemini 3 Pro in the model picker.
- To learn more about the models available in Copilot, see our documentation on models and get started with Copilot today.
