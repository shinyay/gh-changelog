# More direct access to agent session creation across GitHub Mobile

## Overview
We’ve expanded the availability of the New Agent Session option in GitHub Mobile! Previously, users could only start a new agent session from the “Agents”, “Home”, or repository views.

## Detailed Explanation
### Overview
- We’ve expanded the availability of the New Agent Session option in GitHub Mobile! Previously, users could only start a new agent session from the “Agents”, “Home”, or repository views.
- With this update, you can now launch agent sessions from multiple additional touchpoints throughout the app, including:
- Actions report/job view.
- Commit file view.
- Pull request main view.
- File changed view.
- Copilot Chat message overflow menu.
- To streamline your workflow and facilitate quicker task setup, we automatically prepopulate the agent session composer with the relevant shared URL or text message. This enhancement gives you more direct access to agent capabilities, whenever and wherever you need them in the app.
- The new experience is now available on the latest production build of GitHub Mobile on iOS and Android .

## Article Content (cleaned)
We’ve expanded the availability of the **New Agent Session** option in GitHub Mobile! Previously, users could only start a new agent session from the “Agents”, “Home”, or repository views.


With this update, you can now launch agent sessions from multiple additional touchpoints throughout the app, including:


* Actions report/job view.
* Commit file view.
* Pull request main view.
* File changed view.
* Copilot Chat message overflow menu.


To streamline your workflow and facilitate quicker task setup, we automatically prepopulate the agent session composer with the relevant shared URL or text message. This enhancement gives you more direct access to agent capabilities, whenever and wherever you need them in the app.


The new experience is now available on the latest production build of GitHub Mobile on [iOS](https://apps.apple.com/us/app/github/id1477376905) and [Android](https://play.google.com/store/apps/details?id=com.github.android).
