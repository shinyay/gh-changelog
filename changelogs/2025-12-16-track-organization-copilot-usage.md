# Track organization Copilot usage

## Overview
You can now access your organization’s Copilot usage metrics via the Copilot usage APIs .

## Detailed Explanation
### Overview
- You can now access your organization’s Copilot usage metrics via the Copilot usage APIs .
- The APIs contain reports with aggregated and user-specific metrics, including usage statistics for various Copilot features, user engagement data, and feature adoption metrics.
- To see these metrics, the Copilot usage metrics policy must be enabled.
- To enable this setting for an enterprise account, go to the Enterprises page , select your enterprise, and click on the AI Controls tab. In the left sidebar, select Copilot , and scroll down to Metrics . Select Enabled everywhere .
- To enable this setting for a standalone organization account, go to the Organization page , select your organization, and click on the Settings tab. In the left sidebar, select Copilot > Policies . Scroll down to Features > Copilot usage metrics . Select Enabled .
- Organization owners and users with an organization custom role that has the View Organization Copilot Metrics permission can access the API.
- To learn more, see our documentation .

## Article Content (cleaned)
You can now access your organization’s Copilot usage metrics via [the Copilot usage APIs](https://docs.github.com/en/enterprise-cloud@latest/rest/copilot/copilot-usage-metrics?apiVersion=2022-11-28#get-copilot-organization-usage-metrics-for-a-specific-day).


The APIs contain reports with aggregated and user\-specific metrics, including usage statistics for various Copilot features, user engagement data, and feature adoption metrics.


To see these metrics, the **Copilot usage metrics** policy must be enabled.


* To enable this setting for an enterprise account, go to the [Enterprises page](https://github.com/settings/enterprises), select your enterprise, and click on the **AI Controls** tab. In the left sidebar, select **Copilot**, and scroll down to **Metrics**. Select **Enabled everywhere**.
* To enable this setting for a standalone organization account, go to [the Organization page](https://github.com/settings/organizations), select your organization, and click on the **Settings** tab. In the left sidebar, select **Copilot** \> **Policies**. Scroll down to **Features** \> **Copilot usage metrics**. Select **Enabled**.


Organization owners and users with an organization custom role that has the `View Organization Copilot Metrics` permission can access the API.


To learn more, see [our documentation](https://docs.github.com/en/enterprise-cloud@latest/rest/copilot/copilot-usage-metrics?apiVersion=2022-11-28#get-copilot-organization-usage-metrics-for-a-specific-day).
