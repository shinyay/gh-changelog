## Article Content (cleaned)
You can now launch GitHub Copilot prompts directly from select pages on [GitHub Docs](https://docs.github.com).


We’ve introduced dynamic GitHub Copilot prompts, making it easy to move seamlessly from documentation to [Copilot](https://github.com/copilot) so you can get right back to your work. Instead of manually copying and pasting a prompt, you can now launch a prefilled prompt from docs in GitHub Copilot.


Dynamic Copilot prompts are available in three types:


* **Inline prompts**: Send general questions to GitHub Copilot
* **Prompt blocks**: Open a GitHub\-related prompt in GitHub Copilot
* **Prompt blocks with context**: Launch a prompt and a relevant code block in GitHub Copilot


To try out the new dynamic GitHub Copilot prompts, check out the [tutorial on improving code readability](https://docs.github.com/copilot/tutorials/copilot-chat-cookbook/refactor-code/improve-code-readability#example-prompt).
