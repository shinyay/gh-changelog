# Gemini 3 Flash is now in public preview for GitHub Copilot

## Overview
Gemini 3 Flash, ideal for tasks where speed is crucial, is now rolling out in public preview in GitHub Copilot.

## Detailed Explanation
### Overview
- Gemini 3 Flash, ideal for tasks where speed is crucial, is now rolling out in public preview in GitHub Copilot.

### Availability in GitHub Copilot
- Gemini 3 Flash is now rolling out in Copilot Pro, Pro+, Business, and Enterprise.
- You’ll be able to select the model in the Copilot model picker from:
- Visual Studio Code in all modes: chat, ask, edit, agent
- Copilot Chat in github.com
- GitHub Mobile (iOS and Android)
- Rollout will be gradual. If you don’t see the new model yet, check back soon.

### Enabling access
- Enterprise and Business plans : Administrators must enable Gemini 3 Flash in Copilot settings. Once enabled, users in that organization will see the model in the Copilot model picker.
- Pro and Pro+ plans : Users can enable the model by selecting it in the model picker and confirming the one-time prompt.
- Bring your own key : Within Visual Studio Code, select Manage Models from the picker, choose Gemini 3 Flash, and enter your API key when prompted.
