# Reduced pricing for GitHub-hosted runners usage

## Overview
Reduced costs for GitHub-hosted runners are now in effect. We announced this update on December 16, 2025. GitHub users will see price reductions of up to 39% for GitHub-hosted runners, depending on the machines they use.
