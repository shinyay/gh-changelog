# GitHub Mobile: Comment on unchanged lines in pull request files

## Overview
You can now comment anywhere within a changed file on GitHub Mobile. Previously in the Files Changed view, you could only comment on the lines directly surrounding a change. With this new feature you can expand the diff to reveal unchanged code and leave a comment or suggested change wherever it’s needed!

## Detailed Explanation
### Overview
- You can now comment anywhere within a changed file on GitHub Mobile. Previously in the Files Changed view, you could only comment on the lines directly surrounding a change. With this new feature you can expand the diff to reveal unchanged code and leave a comment or suggested change wherever it’s needed!
- Comments added on unchanged lines on github.com are also displayed in GitHub Mobile, so you can review and respond from your phone without missing context.
- This makes it easier to flag changes that should have been made, suggest updates beyond the diff context, and add helpful context for reviewers on unchanged parts of a file.
- The new experience is now available on the latest production build of GitHub Mobile on iOS and Android .
