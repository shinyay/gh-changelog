# Fast mode for Claude Opus 4.6 is now in preview for GitHub Copilot

## Overview
Fast mode for Claude Opus 4.6 is now rolling out in a research preview on GitHub Copilot. This high-speed option delivers output token speeds up to 2.5x faster, while maintaining the same intelligence as Opus 4.6. This release is early and experimental, focused on significantly faster inference.

## Detailed Explanation
### Overview
- Fast mode for Claude Opus 4.6 is now rolling out in a research preview on GitHub Copilot. This high-speed option delivers output token speeds up to 2.5x faster, while maintaining the same intelligence as Opus 4.6. This release is early and experimental, focused on significantly faster inference.

### Availability in GitHub Copilot
- Fast mode for Claude Opus 4.6 will be available to Copilot Enterprise users.
- You’ll be able to select the model in the model picker in:
- Visual Studio Code in all modes: chat, ask, edit, agent
- Copilot CLI
- Rollout will be gradual. Check back soon if you don’t see it yet.

### Enabling access
- Copilot Enterprise plan administrators must enable the Fast mode for Claude Opus 4.6 policy in Copilot settings.
