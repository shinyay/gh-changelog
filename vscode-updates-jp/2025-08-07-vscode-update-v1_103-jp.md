# 2025年7月（バージョン 1.103）

## Overview
_リリース日: 2025-08-07_

このリリースは「モデル/ツール/実行のスケール」に焦点があります。モデル選択肢（GPT-5 系）の拡充、会話の“やり直し”を可能にするチェックポイント、ツール選択 UI と上限回避策の整備など、エージェント運用の現実的な課題に手を入れています。

- チャットで GPT-5 と GPT-5 mini を選択可能に。
- チェックポイントで会話/ワークスペース状態を復元し、試行錯誤を安全に。
- ツールピッカー UI を刷新し、128 ツール上限に対する試験的なグルーピング回避策も追加。
- ターミナル自動承認設定を整理・強化（正規表現フラグなど）。
- Git worktree 対応で、複数ブランチの並行作業を支援。

## Chat
エージェント運用の安全弁（復元）と、スケール（ツール数）への対応が追加されています。

- チャットのモデル選択に GPT-5 / GPT-5 mini が加わり、用途に応じた選択が可能に。
- チェックポイント（設定で有効化）により、以前の状態へ戻してやり直す運用がしやすく。
- ツール選択 UI がツリー構造に刷新され、操作性が改善。
- 128 ツール制限への回避策として、試験的なツールグルーピングが追加。
