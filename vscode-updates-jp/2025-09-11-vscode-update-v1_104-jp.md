# 2025年8月（バージョン 1.104）

## Overview
_リリース日: 2025-09-11_

このリリースは、エージェントの自律性を高めつつ「安全・統制・透明性」を強化する方向性がはっきりしています。モデル自動選択（プレビュー）や、センシティブファイル編集の確認、AGENTS.md の自動取り込み、ターミナル自動承認の明確な opt-in などが代表例です。

- モデルピッカーに Auto（プレビュー）を追加し、利用可能/方針に応じてモデルを選択。
- センシティブなファイル編集では確認を要求する安全策を追加（パターン設定可能）。
- `AGENTS.md` をワークスペース文脈として自動取り込み（実験的、既定有効）。
- 変更ファイル表示/受け入れフローが改善され、エージェント運用時の摩擦を低減。
- ターミナル自動承認は明示的な opt-in とルール管理 UI が整備され、インターネット取得コマンドへの警告も追加。
- #codebase の埋め込みモデル更新など、セマンティック検索の品質/容量面を改善。
- AI 機能全体を無効化/非表示にする設定が追加。

## Chat
“安全に任せる”ためのガードレールと、運用 UX が強化されています。

- Auto モデル選択（プレビュー）で、モデル/倍率の見える化（ホバー等）も含めた運用を支援。
- センシティブファイル編集では確認必須とし、誤編集/悪用リスクを低減。
- `AGENTS.md` の自動取り込みにより、リポジトリ運用ルールを一貫して反映しやすく。
- ツール呼び出しのスキップなど、長いエージェントループ中の“詰まり”を減らす機能が追加。
