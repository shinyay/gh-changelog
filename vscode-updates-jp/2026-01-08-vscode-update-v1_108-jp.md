# 2025年12月（バージョン 1.108）

## Overview
_リリース日: 2026-01-08_

このリリースは“年末の整備/磨き込み”として位置づけられつつ、エージェント運用のカスタマイズ（Agent Skills）やセッション UX、ターミナルの安全性/衛生面、アクセシビリティなど、実運用の品質向上が目立ちます。

- Agent Skills（実験的）: 必要時に読み込める「指示/スクリプト/リソース」をフォルダーとしてまとめる仕組み。
- セッション/履歴: セッション一覧やクイックピックの整備、復元挙動の見直しなど。
- ターミナル: 自動承認ルール/履歴汚染防止/IntelliSense の既定挙動など、安全性と使い勝手を調整。
- アクセシビリティ: Accessible View のストリーミングなど。
